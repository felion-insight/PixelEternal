## 1. 设计理念

### 1.1 三大核心原则

| 原则 | 说明 |
|------|------|
| **件件有身份** | 每件装备都有自己的"基型"（Base Type），不是无脸数值堆 |
| **掉落即兴奋** | 不同稀有度有质变，传奇以上改变玩法 |
| **构建有深度** | 词缀、威能、套装可自由组合，同一职业有 N 种 Build |

### 1.2 对标现代 ARPG

| 借鉴来源 | 借鉴内容 |
|----------|----------|
| **Diablo 4** | 传奇威能（Aspects）、神圣/先祖装备分层 |
| **Path of Exile 2** | 装备基型决定隐式属性、词缀分层（前缀/后缀） |
| **Last Epoch** | 锻造潜力（Forging Potential）、命运预言 |
| **Lost Ark** | 品质切割（Quality）、套装逐步激活 |

### 1.3 装备标签体系

每件装备在鼠标悬停时，玩家一眼能看懂：

```
┌─────────────────────────────┐
│ ⚔️ 龙息战剑  [传说]          │
│ 装备分数：★★★☆☆ (128)       │
│                             │
│ 基型：重剑                   │
│ 单手 · 近战 · 战士亲和        │
│                             │
│ 攻击力：85                   │
│ 暴击率：+15%                 │
│ 暴击伤害：+40%               │
│                             │
│ ──── 随机词缀 ────           │
│ ◆ 对Boss伤害 +12%           │
│ ◆ 击杀回复 3% 生命           │
│ ◇ 火焰伤害 +8~15            │
│                             │
│ ──── 传奇威能 ────           │
│ ★ 龙息：暴击时释放火焰吐息    │
│   (冷却 8秒)                 │
│                             │
│ 套装：龙族 (2/4)             │
│ 职业亲和：战士 (+10%收益)    │
│ 等级需求：15                 │
└─────────────────────────────┘
```

## 2. 装备槽位重构

### 2.1 槽位方案

```
旧：weapon / helmet / chest / legs / boots / necklace / ring / belt
新：mainHand / offHand / head / body / hands / legs / feet / amulet / ring / belt
```

| # | 槽位 | ID | 核心定位 | 主属性 |
|---|------|-----|----------|--------|
| 1 | 主手 | `mainHand` | 攻击来源 | 攻击力 |
| 2 | 副手 | `offHand` | 攻防补充 | 视类型 |
| 3 | 头部 | `head` | 防御+精神 | HP / 防御 |
| 4 | 身体 | `body` | 防御核心 | HP / 防御 |
| 5 | 手部 | `hands` | 攻击辅助 | 攻击 / 攻速 |
| 6 | 腿部 | `legs` | 防御+机动 | HP / 移速 |
| 7 | 足部 | `feet` | 机动核心 | 移速 / 闪避 |
| 8 | 护符 | `amulet` | 特殊效果 | 随机（权重高） |
| 9 | 指环 | `ring` | 攻击辅助 | 暴击 / 资源 |
| 10 | 腰带 | `belt` | 资源管理 | 资源 / 生命 |

> **10个槽位**，比旧版多2个。数量合理：Diablo 4 是 12 槽位（含2戒指），PoE 约 10-12 槽位。

### 2.2 旧体系兼容

旧版8个槽位做如下映射：
- `weapon` → `mainHand`
- `helmet` → `head`
- `chest` → `body`
- `legs` → `legs`
- `boots` → `feet`
- `necklace` → `amulet`
- `ring` → `ring`
- `belt` → `belt`

新增：`offHand`（副手）、`hands`（手部）。旧存档自动补白板占位装备。

## 3. 稀有度体系

### 3.1 六层稀有度

| 稀有度 | ID | 颜色 | 最大词缀数 | 威能 | 掉落阶段 |
|--------|-----|------|-----------|------|----------|
| 普通 | `normal` | 灰 | 0 | 无 | Lv1+ |
| 魔法 | `magic` | 绿 | 1-2 | 无 | Lv1+ |
| 稀有 | `rare` | 蓝 | 3-4 | 无 | Lv5+ |
| 史诗 | `epic` | 紫 | 4-5 | 有概率有 | Lv10+ |
| 传说 | `legendary` | 橙 | 5 | 必有1个 | Lv20+ |
| 神话 | `mythic` | 红 | 5 | 必有2个 | Lv40+（Boss专属掉落） |

### 3.2 稀有度与装备分数

每件装备有一个统一的**装备分数（Gear Score）**，方便快速判断强弱：

```
装备分数 = 基础分（基型×稀有度） + 词缀分 + 威能分

稀有度倍率：普通1.0 / 魔法1.3 / 稀有1.6 / 史诗2.0 / 传说2.5 / 神话3.0
```

### 3.3 稀有度视觉表现

| 稀有度 | 边框 | 光效 | 地面显示 |
|--------|------|------|----------|
| 普通 | 灰框 | 无 | 灰色名字 |
| 魔法 | 绿框 | 微光 | 绿色名字 |
| 稀有 | 蓝框 | 脉冲光 | 蓝色名字+微光柱 |
| 史诗 | 紫框 | 流动紫光 | 紫色光柱 |
| 传说 | 橙框 | 火焰橙光 | 橙色光柱+地面标记 |
| 神话 | 红框 | 雷电红光 | 红色光柱+全屏提示 |

## 4. 装备基型系统（Base Type）

### 4.1 核心理念

**同一槽位、同一等级、同一稀有度，不同基型的装备天差地别。**

举例：同样是 Lv20 传说头盔：
- **重盔（Heavy Helm）**：高防御、高生命，附带减伤
- **轻冠（Light Crown）**：中防御，附带闪避和移速
- **秘法帽（Arcane Hood）**：低防御，附带技能伤害和冷却

### 4.3 防具基型分类

每个防具槽位有 3 种基型：

| feet | 铁靴：高防/减伤 | 皮靴：移速/攻速 | 布鞋：闪避/冷却 |

## 5. 词缀系统（Affix）

### 5.1 前缀/后缀双层制（借鉴 PoE）

每件装备的词缀分为**前缀（Prefix）**和**后缀（Suffix）**：

| 类型 | 倾向 | 示例 |
|------|------|------|
| 前缀 | 攻防数值 | +攻击力、+HP、+防御、+技能伤害 |
| 后缀 | 机制/特殊 | +暴伤、+资源恢复、+冷却缩减、+移速 |

**每件装备稀有度决定前后缀数量**：

| 稀有度 | 前缀数 | 后缀数 |
|--------|--------|--------|
| 魔法 | 1 | 0-1 |
| 稀有 | 1-2 | 1-2 |
| 史诗 | 2-3 | 2 |
| 传说 | 3 | 2 |
| 神话 | 3 | 2 |

### 5.2 词缀分层（Tier）

每条词缀有 Tier 1~5，Tier 越高数值越大：

| Tier | 名称 | 颜色 | 数值范围 | 出现等级 |
|------|------|------|----------|----------|
| T1 | 凡 | 白 | 基准 | Lv1+ |
| T2 | 良 | 绿 | 1.3× | Lv10+ |
| T3 | 湛 | 蓝 | 1.6× | Lv20+ |
| T4 | 炽 | 紫 | 2.0× | Lv35+ |
| T5 | 曜 | 橙 | 2.5× | Lv50+ |

### 5.3 词缀池（完整列表）

#### 前缀池（Prefixes）

| 词缀名 | 效果 | T1值 | T5值 | 可用槽位 |
|--------|------|------|------|----------|
| 强袭 | +攻击力 | 5~10 | 35~50 | 主手/手部/指环 |
| 坚韧 | +最大HP | 20~40 | 120~200 | 头/身/腿/腰带 |
| 壁垒 | +防御 | 5~10 | 25~40 | 头/身/腿/足 |
| 咒术 | +技能伤害% | 5~8% | 15~20% | 主手/头部/护符 |
| 治愈 | +治疗强度% | 8~12% | 20~30% | 护符/指环 |
| 汲取 | +生命偷取% | 1~2% | 4~5% | 主手/手部 |
| 荆棘 | 反伤（固定值） | 3~5 | 15~25 | 身体/腿部 |
| 火焰/冰霜/闪电/暗影 | +元素伤害 | 3~6 | 12~20 | 主手/护符 |

#### 后缀池（Suffixes）

| 词缀名 | 效果 | T1值 | T5值 | 可用槽位 |
|--------|------|------|------|----------|
| 精准 | +暴击率% | 2~4% | 8~12% | 主手/手部/指环 |
| 致命 | +暴击伤害% | 8~15% | 30~50% | 主手/手部/指环 |
| 疾风 | +攻击速度% | 5~8% | 15~25% | 主手/手部/足部 |
| 迅捷 | +移动速度% | 3~5% | 10~15% | 足部/腰带 |
| 闪避 | +闪避率% | 2~4% | 8~12% | 腿部/足部/指环 |
| 冷静 | +冷却缩减% | 3~5% | 10~15% | 头部/护符/腰带 |
| 涌泉 | +资源恢复% | 5~8% | 18~25% | 护符/指环/腰带 |
| 减伤 | 伤害减免% | 1~2% | 4~6% | 身体/头部/护符 |
| 贪婪 | +金币获取% | 5~10% | 20~35% | 腰带/指环 |

### 5.4 词缀权重与职业偏向

不同职业掉落装备时，部分词缀出现概率更高：

| 职业 | 高权重词缀 |
|------|----------|
| 战士 | 坚韧、壁垒、荆棘、减伤 |
| 弓箭手 | 精准、致命、疾风、迅捷 |
| 法师 | 咒术、涌泉、冷静、元素伤害 |
| 刺客 | 精准、致命、闪避、汲取 |

### 5.5 词缀重置系统

使用「洗练石」可重新随机词缀（保持稀有度对应的词缀数量）：

| 消耗 | 效果 |
|------|------|
| 洗练石 ×1 | 随机重置1条后缀 |
| 洗练石 ×3 | 随机重置1条前缀 |
| 洗练石 ×10 + 金币 | 随机升级1条词缀 Tier（最高T4） |

## 6. 传奇威能系统

### 6.1 核心理念

**传说和神话装备的核心价值不是数值，而是改变玩法的威能。**

### 6.2 通用威能池（全职业可用）

| 威能名 | 效果 | 来源 |
|--------|------|------|
| **龙息** | 暴击时释放火焰吐息，对前方锥形造成 80% 攻击力火焰伤害（冷却 8s） | 传说武器 |
| **连锁闪电** | 击杀敌人时释放闪电链，弹跳 3 次，每次 50% 攻击力伤害 | 传说武器 |
| **荆棘光环** | 常驻荆棘光环，每秒对周围敌人造成 20% 防御力的伤害 | 传说身体 |
| **不死鸟** | 受到致命伤害时恢复 40% 最大 HP 并获得 3 秒无敌（冷却 300s） | 传说身体 |
| **幻影步** | 闪避成功后移速 +50%，持续 2 秒（冷却 10s） | 传说足部 |
| **霜冻新星** | 被击中时有 20% 概率释放冰霜新星，冰冻周围敌人 2s | 传说头部 |
| **贪婪** | 怪物掉落金币 +50%，金币拾取范围 +100% | 传说腰带 |
| **血怒** | HP 每降低 10%，伤害 +5% | 传说护符 |

### 6.3 职业专属威能（每个职业2个传说+1个神话）

#### 战士系

| 威能 | 稀有度 | 效果 |
|------|--------|------|
| **战神之怒** | 传说 | 使用职业技能后，4秒内普攻伤害 +60% |
| **不灭之盾** | 传说 | 格挡成功时回复 8% 最大 HP（冷却 6s），并获得 1 层护盾（5% HP） |
| **泰坦之躯** | 神话 | 免疫击退和眩晕，HP >80% 时伤害 +40%，HP <30% 时减伤 +40% |

#### 弓箭手系

| 威能 | 稀有度 | 效果 |
|------|--------|------|
| **鹰眼** | 传说 | 与敌人距离 >150 时，伤害 +35%，暴率 +15% |
| **箭雨** | 传说 | 使用职业技能后，向天空射出箭雨，3秒内对周围敌人造成 300% 总伤 |
| **风行者之魂** | 神话 | 移动时积累风能（每秒 5%），满层时下次攻击必暴击且 +150% 暴伤 |

#### 法师系

| 威能 | 稀有度 | 效果 |
|------|--------|------|
| **奥术洪流** | 传说 | 连续释放 3 个不同元素技能后，下一个技能伤害 +80% 且无消耗 |
| **法力护盾** | 传说 | 消耗法力时 30% 消耗量转化为临时护盾（上限 25% 最大 HP） |
| **元素化身** | 神话 | 法力 >90% 时所有伤害 +50%，且每秒对周围敌人造成 30% 法强的元素伤害 |

#### 刺客系

| 威能 | 稀有度 | 效果 |
|------|--------|------|
| **暗杀** | 传说 | 对满血敌人首次伤害 +150%，必定暴击 |
| **影之舞** | 传说 | 连续暴击 3 次后进入影舞状态 4s：闪避+40%，暴伤+60% |
| **死神降临** | 神话 | 击杀敌人时重置暗影步冷却，并在 2s 内下一个技能伤害 +100% |

### 6.4 威能提取与刻印

借鉴 Diablo 4 的威能系统：

| 操作 | 消耗 | 说明 |
|------|------|------|
| **提取威能** | 金币 ×5000 | 摧毁传说装备，提取其威能存入威能法典（只能使用1次） |
| **刻印威能** | 威能 + 金币 | 将法典中的威能刻印到稀有/史诗装备上，使其变为传说 |
| **威能覆盖** | 新威能 + 金币 | 覆盖已刻印的威能 |

> 武器可刻印 1 个威能，身体可刻印 1 个威能，护符可刻印 1 个威能。最多同时激活 3 个刻印威能。

## 7. 套装系统重做

### 7.1 核心理念改变

```
旧：固定8件套 → 全有或全无，限制了装备多样性
新：2/4件激活 → 更灵活，可混搭不同套装
```

### 7.2 套装激活规则

- 每个套装只需要 **2 件或 4 件** 即可激活效果
- **同一件装备只能属于 1 个套装**（不再像旧版8件占满全身）
- 玩家可混搭：**2+2**（两套2件效果）或 **单体4件**

## 8. 武器系统深度重构

### 8.1 武器 = 基型 + 固有技能 + 词缀 + 传奇威能

```
武器 = {
  baseType: "重剑",           // 基型：决定隐式属性
  implicit: { critDmg: +15% }, // 基型带来的固有效果
  
  attackPower: 85,            // 攻击力（核心数值）
  attackSpeed: 0.8,           // 每秒攻击次数
  range: 50,                  // 攻击距离
  
  weaponSkill: { ... },       // 武器主动技能（替代旧版）
  
  prefixes: [ ... ],          // 前缀词缀（最多3条）
  suffixes: [ ... ],          // 后缀词缀（最多2条）
  
  legendaryPower: { ... },    // 传说/神话威能
  set: "fireheart",           // 所属套装
  
  classAffinity: "warrior",   // 职业亲和
}
```

### 8.2 武器技能系统重做

旧版：武器随机获得一个技能，与职业完全无关。

**新版：武器技能由 基型 + 稀有度 共同决定**

| 基型 | 武器技能方向 |
|------|-------------|
| 重剑 | 猛击类：蓄力重击、裂地斩、破甲 |
| 长剑 | 斩击类：旋风斩、剑气、连斩 |
| 短剑 | 快速类：连刺、疾风斩、反击 |
| 圣剑 | 神圣类：圣光斩、审判、净化 |

| 稀有度 | 技能复杂度 |
|--------|----------|
| 普通 | 无技能（纯普攻） |
| 魔法 | 简单技能（单段伤害） |
| 稀有 | 标准技能（带小效果） |
| 史诗 | 进阶技能（带Buff/Debuff） |
| 传说 | 传奇技能（带独特机制） |
| 神话 | 终极技能（带双效果） |

### 8.3 武器技能 ≠ 职业技能

- **武器技能**：绑定武器，换武器即换技能，有独立冷却（与职业技能冷却互不影响）
- **职业技能**：绑定角色，按等级学习，受 `skillHaste` / `cooldownReduction` 影响

玩家有 **1 个武器技能 + 4 个职业技能**，总共 5 个可释放技能。

### 12.1 掉落层级

| 内容 | 品质上限 | 额外偏向 |
|------|----------|----------|

| 精英怪 | 史诗（紫） | 65%职业亲和 |
| 宝箱 | 史诗（紫） | 无偏向 |
| Boss（每20层） | 传说（橙） | 80%职业亲和 + 1件保底亲和 |
| 终局Boss | 神话（红） | 100%职业亲和 |

### 12.2 智能掉落规则

```
每次掉落执行：
1. 确定掉落槽位 → 均匀随机10个槽位
2. 确定稀有度 → 按稀有度概率表
3. 确定基型 → 按职业偏向权重
4. 生成词缀 → 职业高权重词缀优先
5. 如果传说 → 从威能池抽取（60%通用 / 40%职业专属）
6. 生成装备实例
```
