## 生成优化建议

1. **批量生成工作流**：
```python
# 示例Python代码
base_prompt = "Pixel art icon, retro 16-bit style, blocky pixel clusters, pure black background (#000000), no decorative borders, no text, no stars, minimalistic design, razor-sharp edges, no anti-aliasing, no transparency"

effects = {
"attack": "centered upward-thrusting sword with fiery glow, vibrant red color",
"defense": "centered sturdy shield with metallic sheen, deep blue color",
# ... 其他效果
}

for effect, description in effects.items():
full_prompt = f"{base_prompt}, {description}"
generate_icon(full_prompt)
```

2. **颜色编码系统**：
- 攻击/伤害：红色系
- 防御/保护：蓝色系
- 生命/治疗：绿色系
- 速度/敏捷：青色/黄色系
- 特殊效果：紫色/橙色系

3. **一致性技巧**：
- 所有图标使用相同分辨率生成（推荐1024x1024）
- 后处理统一缩放至64x64（使用NEAREST插值）
- 添加边缘净化处理脚本（消除残留白边）

4. **动态效果关键词**：
- 火焰效果：`fiery glow`, `blazing`, `erupting flames`
- 闪电效果：`electric arcs`, `crackling energy`, `lightning strike`
- 冰冻效果：`icy mist`, `frost spikes`, `chilling aura`
- 治疗效果：`soft glow`, `pulsing light`, `restorative energy`
