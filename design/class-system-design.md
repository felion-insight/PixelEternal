## 1. 设计概述

### 1.1 设计目标

为《Pixel Eternal》设计一套完整的职业系统，支持：
- **4个基础职业**：战士、弓箭手、法师、刺客
- **12个一转职业**：20级解锁，每个基础职业3个分支
- **12个二转职业**：40级解锁，对一转职业的强化和升华
- **PVE多人玩法**：支持团队副本、多人协作

### 1.2 设计原则

1. **团队导向**：每个职业在团队中都有明确价值
2. **机制差异**：职业间不是简单数值差异，而是玩法机制不同
3. **成长曲线**：转职带来质的变化，而非量的提升
4. **平衡性**：12个二转职业各有特色，无绝对强弱

### 1.3 文档结构

```
职业系统设计方案
├── 基础职业（1-19级）
├── 一转职业（20-39级）
├── 二转职业（40-60级）
├── 技能系统
├── 转职系统
└── 数据结构
```

## 2. 设计理念

### 2.1 PVE多人游戏核心定位

| 定位 | 职责 | 代表职业 |
|------|------|----------|
| **坦克（Tank）** | 吸引仇恨，承受伤害，保护团队 | 骑士、守护者、神殿骑士 |
| **治疗（Healer）** | 恢复生命，解除负面状态 | 贤者、先知、圣骑士 |
| **近战DPS（MDPS）** | 近距离高爆发/持续伤害 | 狂战士、影舞者、夜刃 |
| **远程DPS（RDPS）** | 远距离安全输出 | 神射手、巫师、死眼 |
| **辅助（Support）** | 增益队友，削弱敌人 | 守护者、骗术师、先知 |

### 2.2 职业特色设计原则

1. **独特机制**：每个职业有专属资源系统和核心机制
2. **团队技能**：每个职业都有团队增益/减益技能
3. **技能协同**：技能之间有连击、协同效果
4. **成长感知**：每次转职都有明显的玩法改变

## 3. 职业体系架构

### 3.1 职业树总览

```
战士 (Warrior)
├── 骑士 (Knight) → 圣骑士 (Paladin)
├── 狂战士 (Berserker) → 毁灭者 (Destroyer)
└── 守护者 (Guardian) → 神殿骑士 (Temple Knight)

弓箭手 (Archer)
├── 游侠 (Ranger) → 兽王 (Beastmaster)
├── 神射手 (Marksman) → 死眼 (Deadeye)
└── 风行者 (Windrunner) → 幻影 (Phantom)

法师 (Mage)
├── 巫师 (Wizard) → 大魔导师 (Archmage)
├── 贤者 (Sage) → 先知 (Oracle)
└── 术士 (Warlock) → 死灵法师 (Necromancer)

刺客 (Assassin)
├── 影舞者 (Shadowdancer) → 夜刃 (Nightblade)
├── 骗术师 (Trickster) → 幻术师 (Illusionist)
└── 毒师 (Venomancer) → 瘟疫使者 (Plaguebringer)
```

### 3.2 职业数量统计

| 层级 | 数量 | 解锁等级 | 说明 |
|------|------|----------|------|
| 基础职业 | 4 | 1级 | 战士、弓箭手、法师、刺客 |
| 一转职业 | 12 | 20级 | 每个基础职业3个分支 |
| 二转职业 | 12 | 40级 | 每个一转职业1个强化分支 |
| **总计** | **28** | - | 所有可玩职业数量 |

## 9. 转职系统详细设计

### 9.1 转职流程

#### 一转流程（20级）

```
1. 玩家达到20级
   ↓
2. 收到系统提示：可前往主城转职官处转职
   ↓
3. 前往主城「转职官」NPC
   ↓
4. 对话选择「我要转职」
   ↓
5. 显示3个一转职业选项（详细介绍+技能预览）
   ↓
6. 选择职业后，可选择：
   - 直接转职（消耗1000金币）
   - 接受转职试炼（免费，挑战试炼副本）
   ↓
7. 转职完成
   - 获得转职礼包
   - 属性点重置（免费一次）
   - 学习第一个一转技能
```

#### 二转流程（40级）

```
1. 玩家达到40级
   ↓
2. 自动接到二转任务
   ↓
3. 任务：挑战「觉醒试炼」（单人副本）
   ↓
4. 完成试炼后，自动转职
   ↓
5. 转职完成
   - 获得二转礼包
   - 属性点重置（免费一次）
   - 解锁二转技能树
```

### 9.2 转职NPC设计

**转职官**：
- **位置**：主城中央广场（恶魔塔入口旁边）
- **外观**：身穿紫色长袍的神秘老人
- **对话选项**：
  1. 「我要转职」- 进行转职
  2. 「查看职业介绍」- 查看所有职业信息
  3. 「重置属性点」- 消耗钻石重置

## 附录B：设计决策记录

### B.1 为什么一转3个分支？

- 提供足够的选择空间
- 避免选择困难症（4个太多，2个太少）
- 每个分支有明确特色

### B.2 为什么二转只有1个分支？

- 二转是对一转的强化，不是新选择
- 避免玩家选择困难
- 让每个一转职业有明确的终极形态

### B.3 为什么强调团队定位？

- PVE多人游戏需要团队合作
- 每个职业都应有团队价值
- 避免纯 solo 职业设计
