## 一、 核心系统设计概述

《Pixel Eternal》在 v3.0 版本中构建了一个极其深厚且结构化的职业与技能体系。其核心设计亮点包括：
1. **8核心槽位与技能进化**：游戏将技能槽精简为8个核心槽（普攻、核心1、核心2、团队、生存、一转特色、终极技能、流派被动），并在20级和40级通过转职实现技能形态的进化。这种设计既保证了操作的简便性，又提供了极高的成长感。
2. **五大战场定位 (Battle Roles)**：
   - **击破手 (Breaker)**：破防效率 ×2，负责快速削减精英和Boss’的破防条。
   - **强攻手 (Striker)**：破防窗口伤害 +50%，在Boss破防后进行爆发输出。
   - **异常手 (Anomaly)**：状态效率 +40%，状态伤害 +30%，通过施加元素状态制造持续伤害或控制。
   - **支援手 (Support)**：支援效果 +30%，冷却 -20%，提供治疗、护盾、增益或幻象。
   - **速切手 (Switch)**：切换入场 3 秒 Buff，适合快速切入切出进行连招。
3. **独特的资源体系**：每个职业拥有专属的能量资源（战士-怒气/格挡、弓箭手-集中/弹药、法师-法力/元素能量/时光沙漏/灵魂碎片、刺客-影之力/连击点等），极大地丰富了战斗手感和操作逻辑。
4. **状态联动与元素反应 (Status Synergies)**：游戏设计了5大基础状态（灼烧、冻伤、感电、中毒、暗蚀）和2个衍生状态（撕裂、腐蚀），并通过6大联动反应（热震、剧毒火焰、导电冻结、暗影电弧、凋零、混乱）实现了职业间的深度协同和流派构建。

## 三、 状态协同与反应机制评估

游戏内状态协同机制（Status Synergies）是战斗深度的核心。以下是各反应的触发难度、伤害表现与职业契合度评估：

1. **热震 (Thermal Shock)**：`灼烧` + `冻伤` -> 消耗冻伤，触发 **3%最大生命值** 真伤（上限3倍攻击）。
   - **触发难度**：中等。需要火系与冰系技能交替。
   - **伤害表现**：极高（针对高血量Boss）。
   - **最佳职业**：**大魔导师 (Archmage)**。作为元素融合者，其双元素熔合能完美自循环触发热震。

2. **剧毒火焰 (Toxic Flame)**：`灼烧` + `中毒` -> 不消耗状态，**中毒伤害翻倍(2x)**，并扩散中毒。
   - **触发难度**：低。中毒可无限叠加，灼烧极易施加。
   - **伤害表现**：毁灭性的持续伤害 (DOT)。
   - **最佳职业**：**瘟疫使者 (Plaguebringer)** 与 **死灵法师 (Necromancer)**。瘟疫使者能大范围施加猛毒与疫病，配合火系队友或自身火系词缀，能瞬间融化成群怪物。

3. **导电冻结 (Conductive Freeze)**：`冻伤` + `感电` -> 消耗双方，**眩晕 1.5s**，并获得 **20% 易伤** 持续 4s。
   - **触发难度**：中等。
   - **伤害表现**：极佳的控制与爆发前奏。
   - **最佳职业**：**幻影 (Phantom)** 与 **大魔导师 (Archmage)**。提供极强的控场和团队增伤。

4. **暗影电弧 (Shadow Arc)**：`感电` + `暗蚀` -> 不消耗，**电弧链式弹射2个目标**，造成0.6x伤害。
   - **触发难度**：中等。
   - **伤害表现**：极佳的群体清杂能力。
   - **最佳职业**：**夜刃 (Nightblade)** 与 **死灵法师 (Necromancer)**。夜刃在暗影领域中配合感电能造成恐怖的连锁割草效果。

5. **凋零 (Wither)**：`中毒` + `暗蚀` -> 不消耗，**降低敌人 25% 攻击力** 持续 4s。
   - **触发难度**：低。
   - **伤害表现**：防御型反应，大幅提升生存能力。
   - **最佳职业**：**瘟疫使者 (Plaguebringer)** 与 **死灵法师 (Necromancer)**。

6. **混乱 (Chaos)**：任意 **3种及以上状态** -> 消耗所有状态，**眩晕 1s** 并触发 **0.5x 范围爆炸伤害**。
   - **触发难度**：高。需要多职业配合或特定流派。
   - **伤害表现**：极高爆发与群体强控。
   - **最佳职业**：多职业组队，或拥有极多状态词缀的史诗装备流派。

## 四、 综合平衡性与设计建议

### 1. 优势与闪光点
- **技能形态高度差异化**：从瞬发、弹丸、召唤、场域到冲锋和传送，技能的物理行为（Entity Types）极为丰富，极大地提升了 Canvas 2D 战斗的空间感和操作上限。
- **数值梯度合理**：普攻在 1.0x 左右，核心技能在 1.5x - 3.0x，终极技能和生存爆发技能在 5.0x - 15.0x，甚至骗术师的 `phantom_array` 达到了恐怖的 **12.0x**，毒师的 `withering` 达到了 **25.0x**，体现了高难度操作或长CD技能的极高价值。
- **转职进化逻辑清晰**：一转和二转通过技能进化，不仅提升了伤害倍率，还彻底改变了技能形态（例如：盾击进化为圣光盾击，再进化为审判之盾，附带暗蚀状态），成长反馈极强。

### 2. 潜在问题与优化建议
- **部分二转职业技能数量不均**：例如，`archmage` 只有3个专属技能，而 `plaguebringer` 有7个，`nightblade` 有7个。建议在后续开发中，为 `archmage`、`beastmaster`、`phantom` 等职业补充 1-2 个专属技能，使其技能池达到 5-6 个，保证流派的丰富度。
- **部分超高倍率技能的平衡性**：毒师的 `withering`（枯萎）拥有 25.0x 的超高伤害倍率，虽然是 50s 的长CD生存/爆发技能，但在特定装备冷却缩减或叠毒流派下，可能会造成Boss被瞬间秒杀，失去破防机制的博弈乐趣。建议引入「状态层数消耗」或「破防状态下倍率生效」的限制。
- **召唤单位 (Summon) 的 AI 与成长挂钩**：术士和兽王高度依赖召唤单位（狼、骷髅、亡灵军团）。需要确保召唤单位的生命值和攻击力与玩家的面板属性（如魔法攻击力、生命值）有清晰的等比挂钩公式，否则在游戏后期高难度副本中，召唤物可能会沦为脆皮或刮痧工具。
