## 1. 设计理念

### 1.1 借鉴晶核的核心设计

| 晶核系统 | 借鉴要点 | Pixel Eternal 化 |
|----------|----------|-------------------|
| **强化段保底** | +11起失败降级，+16失败破损，保护符防降级 | 同样设计，增加塔内特色材料 |
| **装备多阶段进阶** | 50粉→50史诗→55粉→55史诗→60史诗 | 每20级一个装备阶段，材料来自对应副本 |
| **不同副本产出不同材料** | 每日刷材料副本，周常刷团本 | 日常/周常/深渊/裂隙 各有分工 |
| **徽记镶嵌** | 装备孔位嵌入徽记提升属性 | 3孔徽记系统（攻击/防御/特殊） |
| **强化转移** | 旧装备强化等级可转移，投资不浪费 | 100%转移（只消耗少量金币） |

### 1.2 核心原则

> **每天有事做，每周有追求，终局有挑战。**

- 短周期：日常副本 → 材料积累（每天30分钟）
- 中周期：周常副本 → 突破材料（每周2-3次）
- 长周期：终局副本 → 毕业装备（持续养成）

## 2. 副本体系总览

### 2.1 副本分类

```
副本体系
├── 恶魔塔（不变）            → 主推进度，装备掉落
├── 材料秘境（不变）          → Boss战，打造材料
│
├── 【新增】日常副本 × 3      → 强化材料、徽记
├── 【新增】周常副本 × 3      → 突破石、保护符、神话碎片
├── 【新增】深渊副本 × 5      → 高品装备掉落、徽记碎片
├── 【新增】裂隙副本（随机）  → 词缀随机，奖励随机翻倍
├── 【新增】远古遗迹 × 3      → 神话材料、职业专属材料
└── 【预留】团队Raid × 3     → 未来多人联机时开放
```

## 4. 周常副本设计

### 4.1 守护者试炼（每周3次，Lv20+）

```
连续5波怪物防守战
  │
  ├── 波1：普通怪 × 8
  ├── 波2：精英怪 × 1 + 普通怪 × 6
  ├── 波3：远程怪 + 近战怪混合 × 10
  ├── 波4：精英怪 × 2（不同种族）
  └── 波5：Boss「守护者之魂」

奖励（按波次完成数）：
- 3波通关：中级强化石 ×3 + 金币 ×1000
- 4波通关：高级强化石 ×2 + 保护符 ×1
- 5波通关：突破石 ×1 + 保护符 ×2 + 徽记碎片 ×3
```

### 4.2 远古龙巢（每周2次，Lv35+）

| 属性 | 说明 |
|------|------|
| 场景 | 龙族巢穴，三阶段Boss战 |
| 阶段1 | 幼龙群（5只），考验AOE |
| 阶段2 | 青年龙 ×2，考验走位和单体爆发 |
| 阶段3 | 远古巨龙，全屏技能+阶段狂暴 |
| 掉落 | 高级强化石 ×5、突破石 ×2、龙鳞 ×1~3 |
| 首通奖励 | 传说装备箱 ×1（Lv35传说随机） |

**龙鳞**：稀有材料，用于装备突破（突破+15→+16需要龙鳞）。

### 4.3 终焉之塔（每周1次，Lv50+）

| 属性 | 说明 |
|------|------|
| 场景 | 爬塔式，10层连续挑战 |
| 机制 | 每层词缀随机（同裂隙副本词缀），不可回血（每5层回复30%） |
| 层5 | 精英Boss ×1 |
| 层10 | 终焉之主（最终Boss） |
| 掉落 | 突破石 ×3、保护符 ×3、神话碎片 ×1~3 |
| 全通奖励 | 神话碎片 ×5 + 职业技能强化石 ×2 |

## 5. 终局副本设计

### 5.1 深渊副本（每日3次/难度）

三层深渊对应不同难度和产出：

| 难度 | 等级 | 怪物强度 | 掉落重点 |
|------|------|----------|----------|
| 浅层 | Lv20 | 普通~精英 | 稀有/史诗装备、初级徽记碎片 |
| 中层 | Lv30 | 精英主体 | 史诗装备、中级徽记碎片 |
| 深层 | Lv40 | 精英+Boss | 史诗/传说装备、高级徽记碎片 |
| 地狱 | Lv55 | Boss连战 | 传说装备、完整徽记（概率） |

**深渊特色**：击杀怪物积累「深渊能量」，满槽触发「深渊之门」→ 额外Boss+额外掉落。

### 5.2 裂隙副本（随机刷新）

| 属性 | 说明 |
|------|------|
| 生成 | 主城随机出现「时空裂隙」，持续2小时 |
| 词缀 | 每次随机2~3个词缀（影响战斗+奖励） |
| 消耗 | 裂隙石 ×1（日常副本/恶魔塔概率掉落） |
| 奖励 | 基础掉落 ×2（含词缀加成） |

**随机词缀表**：

| 词缀 | 效果 | 奖励加成 |
|------|------|----------|
| 狂暴 | 怪物攻速+30% | 金币+40% |
| 铁壁 | 怪物防御+50% | 装备掉落率+30% |
| 加速 | 玩家和怪物移速+50% | 材料掉落+50% |
| 精英 | 精英怪出现率翻倍 | 稀有度+1级 |
| 诅咒 | 玩家持续掉血（每秒1%） | 所有奖励 ×2 |
| 迷雾 | 视野缩小50% | 经验+100% |
| 脆弱 | 怪物HP-40%、攻击+80% | 材料+30% |
| 宝藏 | 怪物数量-30% | Boss必掉传说 |

### 5.3 远古遗迹（每周1次/遗迹）

三个远古遗迹对应职业专属材料产出：

| 遗迹 | 等级 | 职业偏向 | 专属掉落 |
|------|------|----------|----------|
| 第一遗迹·战神殿 | Lv45 | 战士/刺客 | 战士徽记碎片、近战专属材料 |
| 第二遗迹·魔塔 | Lv55 | 法师/弓箭 | 法师徽记碎片、远程专属材料 |
| 最终遗迹·起源 | Lv60 | 全职业 | 神话碎片、职业觉醒石 |

**觉醒石**：用于装备觉醒（详情见第9章）。

### 5.4 【预留】团队Raid（未来多人）

## 6. 装备成长路线

### 6.1 阶段划分（借鉴晶核多阶段进阶）

```
阶段 0：Lv 1-19 → 基础装备
   │  有什么穿什么，强化到+5即可
   │
阶段 1：Lv20-29 → 精良阶段
   │  推进恶魔塔，收集精良/史诗装备
   │  全身强化+8，开始收集套装
   │
阶段 2：Lv30-39 → 史诗阶段
   │  深渊浅层/中层次装备
   │  全身强化+10，2件套装激活
   │  开始镶嵌徽记（1孔）
   │
阶段 3：Lv40-49 → 传说阶段
   │  深渊深层刷传说装备
   │  全身强化+12，4件套装激活
│  徽记升级到紫色
│
阶段 4：Lv50-55 → 突破阶段
   │  远古遗迹获取专属材料
   │  武器强化+15，防具+13
│  徽记全金
│  装备第一次突破（突破1）
   │
阶段 5：Lv55-60 → 觉醒阶段
   │  终焉之塔+远古遗迹毕业装备
   │  全身强化+15~+18
   │  装备第二次突破（突破2）+ 觉醒
   │  神话装备开始合成
   │
终局：Lv60 → 神话阶段
   │  试炼塔无限爬塔
   │  全身强化+20
   │  神话装备全套+神话徽记
   │  装备觉醒满级
```

### 6.2 各阶段毕业战力参考

## 8. 强化系统重做

### 8.1 强化等级段（借鉴晶核的分段设计）

```
┌──────────────────────────────────────────────────────┐
│  强化段          范围    失败惩罚    保护方式         │
├──────────────────────────────────────────────────────┤
│  安全段         +0~+5   无惩罚      无需求            │
│  稳定段         +6~+10  无降级      无需求            │
```

| +16→+17 | 15% | 25000 | 顶级×3 | - |
| +17→+18 | 12% | 30000 | 顶级×4 | - |
| +18→+19 | 10% | 40000 | 顶级×4 | 突破石×3 |
| +19→+20 | 8% | 50000 | 顶级×5 | 神话碎片×3 |

### 8.3 强化保底机制

| 机制 | 说明 |
|------|------|
| **单件保底** | 连续失败3次后，第4次成功率翻倍（每件独立计数） |
| **失败累计** | 每失败1次，积攒1点「强化能量」，30点可兑换必定成功1次 |
| **强化券** | 活动/周常副本概率掉落，使用后单次强化成功率+20% |

### 8.4 强化转移

装备更换时，强化等级可无损转移：

| 转移方式 | 消耗 | 转移比例 |
|----------|------|----------|
| 同品质同部位 | 金币×1000 | 100%（强化等级完全继承） |
| 低品质→高品质 | 金币×3000 | 100% |
| 高品质→低品质 | 金币×1000 | 100% |
| 武器→武器 | 金币×2000 | 100% |
| 跨部位 | ❌ 不可转移 | - |
| 徽记转移 | 金币×500/个 | 100%（徽记完全继承） |

## 9. 装备突破与觉醒

### 9.1 装备突破（借鉴晶核装备进阶）

装备达到特定强化等级后，可以进行「突破」：

| 突破阶段 | 强化要求 | 所需材料 | 效果 |
|----------|----------|----------|------|
| 突破1 | 强化+10 | 突破石×2 + 职业精华×5 + 金币×10000 | 全属性+15%，解锁第2徽记孔 |
| 突破2 | 强化+15 | 突破石×4 + 龙鳞×2 + 职业精华×10 + 金币×30000 | 全属性+25%，解锁第3徽记孔 |
| 觉醒 | 强化+18 | 觉醒石×1 + 神话碎片×5 + 金币×50000 | 全属性+40%，激活觉醒技能（装备专属） |

**突破不可逆**，但强化等级保留（强化+15突破后仍是+15，只是多了解锁效果）。

### 9.2 装备觉醒技能（每个部位1个觉醒效果）

| 部位 | 觉醒效果示例 | 说明 |
|------|-------------|------|
| 武器 | 攻击时3%概率释放500%伤害一击 | 类似"灭绝一击" |
| 头盔 | 被击中时5%概率重置1个技能冷却 | 战斗中的意外惊喜 |
| 身体 | HP归零时保留1HP并获得4秒无敌（冷却300s） | 第二条命 |
| 手部 | 使用技能后2秒内下次普攻伤害+200% | 技能接普攻 |
| 腿部 | 闪避成功后4秒内移速+40%、闪避+20% | 风筝强化 |
| 足部 | 移动每秒积累1%伤害加成（上限30%，停止移动后3秒重置） | 鼓励走A |
| 护符 | 每30秒自动解除一次负面效果 | 净化效果 |
| 指环 | 暴击时3%概率触发暴伤×3 | 超级暴击 |
| 腰带 | 拾取金币时3%概率获得双倍 | 刷金辅助 |
| 副手 | 格挡成功回复5%最大HP（冷却8s） | 坦克续航 |

### 9.3 觉醒效果可洗练

使用「觉醒重置石」可重新随机觉醒效果（远古遗迹最终遗迹掉落）。

## 10. 徽记系统

### 10.1 设计理念

借鉴晶核的徽记系统：装备带有徽记孔位，可通过嵌入徽记获得额外属性。徽记可在强化转移时继承。

### 10.2 徽记类型

| 类型 | 孔位颜色 | 可嵌入属性 |
|------|----------|----------|
| 攻击型 | 🔴 红色 | 攻击力、暴击率、暴击伤害、技能伤害 |
| 防御型 | 🔵 蓝色 | HP、防御、减伤、闪避 |
| 特殊型 | 🟡 金色 | 冷却缩减、资源回复、移速、攻速、吸血 |

### 10.3 徽记品质与升级

| 品质 | 属性值 | 合成需求 | 升级消耗 |
|------|--------|----------|----------|
| 白色 | 基准×1 | 碎片×5 | - |
| 绿色 | 基准×1.5 | 碎片×10 + 合成石×1 | 金币×2000 |
| 蓝色 | 基准×2 | 碎片×20 + 合成石×2 | 金币×5000 |
| 紫色 | 基准×3 | 碎片×40 + 合成石×4 | 金币×10000 |
| 金色 | 基准×5 + 额外特效 | 碎片×80 + 合成石×8 | 金币×20000 |

**金色徽记额外特效示例**：
| 徽记 | 金色特效 |
|------|----------|
| 攻击·暴怒 | 暴击时3%概率提升20%攻击力，持续6秒 |
| 防御·铁壁 | HP<30%时减伤额外+15% |
| 特殊·疾风 | 闪避成功后1秒内无敌 |

### 10.4 孔位解锁

| 孔位 | 解锁条件 |
|------|----------|
| 第1孔 | 装备天生自带（绿色及以上品质） |
| 第2孔 | 装备突破1后解锁 |
| 第3孔 | 装备突破2后解锁 |

> 每个装备最多3个徽记孔（红色×1 + 蓝色×1 + 金色×1）。
