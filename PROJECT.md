# Pixel Eternal - 像素永恒

## 项目概述

Pixel Eternal 是一款基于 HTML5 Canvas 的像素风格 RPG 游戏，玩家在主城中探索、在恶魔塔中战斗、在材料秘境中挑战 Boss，通过装备强化、炼金和打造系统提升战力。

---

## 游戏架构

### 技术栈
- **前端**：HTML5 + CSS + 原生 JavaScript
- **渲染**：Canvas 2D
- **存储**：LocalStorage（支持存档导出/导入，使用 LZ-String 压缩）

## 场景系统

### 主城 (Town)
玩家出生点和休整区域，包含以下可交互建筑：

| 建筑       | 位置           | 功能                               |
|------------|----------------|------------------------------------|
| 恶魔塔入口 | 画面中央       | 进入恶魔塔进行闯关战斗             |
| 铁匠铺     | 左上 (200,200) | 装备强化、精炼、打造               |
| 商店       | 右上           | 购买装备、消耗品随机箱、出售物品   |
| 训练场     | 左下           | 生成训练假人/怪物，测试战斗        |
| 材料秘境   | 右下           | 选择秘境，消耗许可证进入 Boss 战   |

### 恶魔塔 (Tower)
 Roguelike 爬塔模式，每层随机房间类型：
- **战斗房间**：消灭所有怪物进入下一层
- **宝箱房间**：开启宝箱获得奖励
- **休整房间**：使用篝火/医疗箱/疗愈之泉/神圣祭坛恢复
- **炼金房间**：使用炼药台炼制自定义药水

### 训练场 (Training Ground)
可生成训练假人或指定怪物，用于测试装备与技能。支持无敌模式。

### 材料秘境 (Dungeon)
独立 Boss 战场景，需消耗专属攻略许可证进入，击败 Boss 后获得打造材料、大量金币和经验。

## 物品系统

### 物品分类
| 类型   | 说明                         | 背包页签 |
|--------|------------------------------|----------|
| 装备   | 武器、头盔、胸甲等 8 个部位 | 装备     |
| 材料   | 炼金材料、打造材料           | 材料     |
| 消耗品 | 药水、许可证、图纸等         | 消耗品   |

### 装备
- **部位**：武器、头盔、胸甲、腿甲、足具、项链、指环、腰带
- **品质**：普通、稀有、精良、史诗、传说
- **来源**：怪物掉落、宝箱、商店、打造（部分专属装备）

### 打造材料
用于铁匠铺打造装备，主要来自秘境 Boss 掉落：
- 普通：粗糙铁块、精炼钢锭
- 稀有：魔法水晶碎片、龙鳞碎片
- 精良：秘银矿石、星辰碎片
- 史诗：远古龙骨、神之碎片
- 传说：混沌核心、永恒之石

### 消耗品子类型
| 子类型   | 说明                           |
|----------|--------------------------------|
| 药水     | 临时增益或恢复生命             |
| 神圣十字架 | 死亡时复活，恢复满血+3秒无敌 |
| 攻略许可证 | 进入对应秘境的凭证           |
| 打造配方 | 铁匠铺打造装备的图纸           |
| 背包扩容 | 扩展背包容量                   |

## 战斗系统

### 玩家操作
- **移动**：WASD
- **攻击**：J
- **冲刺**：K（短暂无敌，冷却 1 秒）
- **武器技能**：Q
- **交互**：E

### 战力系统
战力综合等级、装备、属性（攻击、防御、生命、暴击等）计算，在左侧状态栏实时显示。装备变动时会出现战力变化提示。

### 死亡与复活
- 恶魔塔/秘境中死亡：失去本次获得的物品和金币
- 若拥有**神圣十字架**，可选择消耗 1 个复活（满血 + 3 秒无敌）
- 无神圣十字架时返回主城

---

## 铁匠铺系统

### 强化
- 消耗金币提升装备属性（每级 +10%）
- 成功率随强化等级下降（+0→+1 为 100%，+9→+10 为 10%）
- 失败仅消耗金币，装备等级不变

### 精炼
- 使用相同装备作为材料进行精炼，提升品质

### 打造
- 需要：打造配方 + 指定基础装备 + 打造材料
- 打造装备不可在商店购买或恶魔塔掉落
- 配方可在商店随机箱中刷新获得

---

## 炼金系统

- 最多选择 5 个炼金材料进行炼制
- 材料品质影响词条保留概率（普通 40%～传说 80%）
- 成品药水品质由材料最高品质决定
- 相同词条会叠加

---

## 商店系统

- **装备页签**：定向位（传说/史诗/精良）+ 随机装备列表
- **消耗品页签**：随机礼箱，开出药水、神圣十字架、许可证、图纸等
- **出售页签**：出售背包中的装备、材料、消耗品
- 刷新商店需消耗金币（固定 150）

---

## 秘境系统

### 秘境列表
| 秘境       | 难度 | 推荐战力 | 可掉落材料品质     |
|------------|------|----------|--------------------|
| 初级材料秘境 | 简单 | 500      | 普通、稀有         |
| 中级材料秘境 | 普通 | 1500     | 稀有、精良         |
| 高级材料秘境 | 困难 | 3000     | 精良、史诗         |
| 大师材料秘境 | 极难 | 5000     | 史诗、传说         |

### 攻略许可证
- 每个秘境有专属许可证
- 获取途径：商店随机箱、恶魔塔怪物掉落
- 进入秘境消耗 1 张对应许可证，退出不返还

## 开发说明

### 运行游戏

由于浏览器安全限制，不能直接打开 HTML 文件（`file://` 协议）。请使用本地服务器运行：

**方法 1：使用启动脚本（推荐）**
```bash
python3 start-server.py
```

**方法 2：使用 Python 内置服务器**
```bash
python3 -m http.server 8000
```
然后访问 `http://localhost:8000/index.html`

**方法 3：使用 Shell 脚本**
```bash
./start-server.sh
```

### 其他说明

- 开发者模式（F1）提供：等级/金币/装备/材料/药水/房间控制等调试功能
- 存档支持导出/导入，便于测试和备份
- 装备贴图存放在 `ArtAssets/` 目录，按装备名称对应
