## 1. 技能框架

### 1.1 技能槽位

| 等级 | 技能类型 | 说明 |
|------|----------|------|
| 1 | 基础攻击 | 普攻（跟随职业变化） |
| 3 | 职业技能1 | 核心输出/机制技能 |
| 6 | 职业技能2 | 辅助输出/位移技能 |
| 10 | 团队技能 | 团队增益/减益（当前单人→范围效果） |
| 15 | 职业技能3 | 核心机制/生存技能 |
| 20 | 一转觉醒 | 一转后获得 |
| 25 | 一转技能1 | 一转专属 |
| 30 | 一转技能2 | 一转专属 |
| 35 | 一转技能3 | 一转专属 |
| 40 | 二转觉醒 | 二转后获得 |
| 45 | 二转技能1 | 二转专属 |
| 50 | 二转技能2 | 二转专属 |
| 55 | 终极技能 | 职业终极技能（长冷却） |
| 60 | 传说技能 | 满级传说被动 |

### 1.2 技能类型说明

| 标识 | 含义 |
|------|------|
| ⚔️ 主动 | 玩家按键释放 |
| 🔒 被动 | 永久生效，无需释放 |
| 🌀 触发 | 满足条件自动触发 |
| 👥 团队 | 影响团队成员 |
| ⭐ 终极 | 长冷却高效果 |
| 🏆 传说 | 满级被动 |

### 1.3 资源类型

| 职业系 | 资源 | 上限 | 恢复方式 |
|--------|------|------|----------|
| 战士系 | 怒气 | 100 | 普攻+8，受击+5，脱战每秒-3 |
| 弓箭手系 | 集中值 | 100 | 每秒+4，暴击+10 |
| 法师系 | 法力 | 200 | 每秒+6 |
| 刺客系 | 能量 | 120 | 每秒+12 |

|----------|----------|------|
| 普攻 | 100%/s DPS | 基准 |
| 小技能（短CD 4-8s） | 120-180% | 单体 |
| 小技能AOE | 80-120% | 群体 |
| 中技能（中CD 10-20s） | 200-300% | 单体 |
| 中技能AOE | 150-250% | 群体 |
| 大技能（长CD 25-40s） | 300-500% | 单体 |
| 大技能AOE | 250-400% | 群体 |

| 召唤物 | 总DPS=标准×1.2 | 持续存在 |
