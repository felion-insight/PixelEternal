## 1. 现状分析

### 1.1 当前成长曲线

```
经验公式：expNeeded = 20 × level²
属性成长：HP +20/级，攻击 +2/级（Lv21起额外+4.5/级）

Lv1:  HP=100, ATK=10,  EXP=0/20
Lv20: HP=480, ATK=48,  EXP=0/8000
Lv40: HP=880, ATK=178, EXP=0/32000
Lv60: HP=1280,ATK=398, EXP=0/72000
```

**问题**：纯线性数值成长，没有职业差异，没有里程碑。

### 1.2 当前主城设施

| 建筑 | 位置 | 功能 |
|------|------|------|
| 恶魔塔入口 | 中央 | 进入爬塔 |
| 铁匠铺 | 左上 | 强化、精炼、打造 |
| 商店 | 右上 | 买装备、消耗品、出售 |
| 训练场 | 左下 | 测试战斗 |
| 材料秘境 | 右下 | Boss战获取材料 |

### 1.3 当前操作

```
WASD → 移动
J → 攻击
K → 冲刺
Q → 武器技能
E → 交互
```

### 1.4 当前缺失

- ❌ 职业选择/切换
- ❌ 职业技能绑定
- ❌ 资源条（怒气/集中/法力/能量）
- ❌ 转职流程
- ❌ 天赋/技能升级
- ❌ 多人副本
- ❌ 玩家差异化成长

## 2. 成长曲线重设计

### 2.1 经验公式调整

旧公式过于陡峭（Lv60需要72000经验），新设计偏向平滑：

```
旧：expNeeded = 20 × level²
新：expNeeded = 100 + 15 × level²

阶段节奏：
Lv1:   100/115
Lv20:  100/6100   （旧：8000）
Lv40:  100/24100  （旧：32000）
Lv60:  100/54100  （旧：72000）
```

**变化**：前期节奏加快（快速进入一转），中后期平缓增长。

### 2.2 经验获取调整

| 来源 | 旧值 | 新值 | 说明 |
|------|------|------|------|
| 普通怪 | 怪物配置值 | ×1.2 | 略微增加 |
| 精英怪 | 普通怪 ×2 | 普通怪 ×3 | 精英更有价值 |
| Boss | 普通怪 ×10 | 普通怪 ×10 + 固定奖励 | 确保Boss战升级感 |
| 精英房通关 | 无 | +当前层怪物经验×5 | 新增奖励 |

### 2.3 属性成长改为职业驱动

```
旧：统一成长（HP+20/级, ATK+2/级）
新：按职业分化
```

| 职业 | HP/级 | ATK/级 | DEF/级 | 特色 |
|------|-------|--------|--------|------|
| 战士 | 25 | 2.2 | 1.0 | 最高HP |
| 弓箭手 | 18 | 2.5 | 0.5 | 高攻击 |
| 法师 | 15 | 2.8 | 0.3 | 最高攻击 |
| 刺客 | 18 | 2.5 | 0.5 | 高速度 |

> 一转后乘以职业修正系数，二转后再次修正。具体数值见 `class-system-design.md`。

### 2.4 升级里程碑

| 等级 | 事件 | 说明 |
|------|------|------|
| 1 | 选择基础职业 | 获得基础技能 |
| 3 | 解锁技能2 | 获得第二个职业技能 |
| 6 | 解锁技能3 | 获得第三个职业技能 |
| 10 | 解锁团队技能 | 获得团队增益/减益技能 |
| 15 | 解锁技能4 | 获得第四个职业技能 |
| 20 | 🔑 一转 | 选择进阶职业，获得觉醒技能 |
| 25/30/35 | 一转技能 | 解锁一转专属技能 |
| 40 | 🔑 二转 | 完成觉醒试炼，获得二转觉醒 |
| 45/50 | 二转技能 | 解锁二转专属技能 |
| 55 | ⭐ 终极技能 | 解锁职业终极技能 |
| 60 | 🏆 传说被动 | 解锁满级传说被动 |

## 3. 职业选择与新手流程

### 3.1 新玩家首次进入

```
游戏启动
  ↓
标题画面（Pixel Eternal Logo）
  ↓
点击「新游戏」
  ↓
职业选择界面（4个基础职业卡片）
  ├── 战士：近战·攻守兼备·推荐新手
  ├── 弓箭手：远程·风筝输出·操作灵活
  ├── 法师：远程·元素轰炸·高伤低防
  └── 刺客：近战·爆发输出·操作进阶
  ↓
查看职业详情（技能预览+属性+介绍）
  ↓
确认选择
  ↓
角色命名（可选，默认"冒险者"）
  ↓
进入游戏 → 主城出生点
  ↓
引导任务：与转职官对话 → 了解职业系统
  ↓
自由探索
```

### 3.2 老玩家（已有存档）

- 旧存档读取后，**自动弹出职业选择界面**（一次性）
- 选择后不可更改（除非消耗「转职石」重置为基础职业）
- 旧存档的装备/金币/等级全部保留

### 3.3 存档兼容策略

```javascript
// 旧存档（无 classData 字段）→ 自动弹出职业选择
if (!saveData.player.classData) {
  showClassSelectionModal(); // 必须选择才能继续游戏
}

// 选完职业后立即重算属性
player.updateStats(); // 会包含职业基础属性
```

## 4. 转职系统接入

### 4.1 一转流程（Lv20）

```
达到Lv20
  ↓
屏幕上方弹出提示：⚡ 已达到20级，可以转职！
  ↓
回到主城，与「转职官」对话（头顶感叹号）
  ↓
转职官对话树：
  「年轻的勇士，你已经具备了进阶的资格」  
  → [查看职业介绍] → 展示3个一转职业详情
  → [我要转职！] → 选择界面
  ↓
选择一转职业：
  ├── 消耗1000金币（直接转职）
  └── 免费挑战试炼（进入独立试炼房间）
  ↓
转职成功动画
  ↓
获得转职奖励：
  - 新职业技能 ×1（觉醒技能）
  - 属性重算
  - 属性点/技能点重置券 ×1
  - 转职称号
```

### 4.2 二转流程（Lv40）

```
达到Lv40
  ↓
屏幕上方提示：⚡ 已达到40级，可以觉醒！
  ↓
自动接到「觉醒试炼」任务
  ↓
前往「觉醒之门」（主城新增建筑）
  ↓
进入独立Boss战（根据一转职业生成对应Boss）
  ↓
击败Boss后：
  - 获得二转觉醒技能
  - 解锁终极技能树
  - 获得二转称号
  - 获得二转礼包（传说装备 ×1、金币 ×10000）
```

### 4.3 转职试炼设计

每个一转职业有独立的试炼内容，难度适中：

| 一转职业 | 试炼Boss | 特点 |
|----------|----------|------|
| 骑士 | 堕落圣骑士 | 高防、会回血 |
| 狂战士 | 狂暴战鬼 | 低血高攻，需要爆发击杀 |
| 守护者 | 守护者之影 | 有护盾，需要持续输出 |
| 游侠 | 兽群之主 | 召唤小怪，测试AOE |
| 神射手 | 暗影射手 | 远程风筝，会隐身 |
| 风行者 | 风暴之灵 | 高速移动，需要命中 |
| 巫师 | 元素之核 | 元素护盾切换 |
| 贤者 | 黑暗主教 | 会治疗，需要打断 |
| 术士 | 亡魂集合体 | 召唤亡灵，测试DOT |
| 影舞者 | 暗影镜像 | 复制玩家技能 |
| 骗术师 | 幻象大师 | 分身迷惑 |
| 毒师 | 剧毒巨蛛 | 毒伤环境 |

## 6. 天赋系统（新增）

### 6.1 设计理念

天赋系统为二转后的终局内容，提供**同职业不同Build**的差异化路线。

- **解锁条件**：Lv40 完成二转后解锁
- **天赋点数**：Lv40~60 每级获得1点，共20点
- **天赋树结构**：三个分支（每分支深度6层）

### 6.2 天赋树示例：骑士 → 圣骑士

```
         ┌─────────────┐
         │  圣光之源    │ (起点)
         └──┬───┬───┬──┘
            │   │   │
   ┌────────┘   │   └────────┐
   ▼            ▼            ▼
 防御分支     治疗分支      攻击分支
 ────────    ────────     ────────
 L1 坚韧+5%  L1 治疗+10%  L1 神圣打击+15%
 L2 格挡+8%  L2 光环范围+30% L2 审判伤害+20%
 L3 受击回怒 L3 净化附带治疗 L3 神圣能量上限+1
 L4 盾墙+2s  L4 复活冷却-60s L4 光明审判必暴
 L5 荆棘翻倍 L5 队友治疗+30% L5 暴击+20%
 L6 🔑圣盾壁垒 L6 🔑生命之泉 L6 🔑神圣复仇
```

每分支第6层是**终极天赋**（只能选1个）：

| 分支 | 终极天赋 | 效果 |
|------|----------|------|
| 防御 | 圣盾壁垒 | 受到致命伤害时获得8秒无敌（冷却300s） |
| 治疗 | 生命之泉 | 神圣之盾每秒额外恢复10%HP |
| 攻击 | 神圣复仇 | 光明审判击杀目标后重置冷却 |

### 6.3 天赋点分配规则

- 每级获得1天赋点
- 天赋可随时免费重置（主城「转职官」处）
- 同分支至少投入5点才能点亮终极天赋
- 三个分支可自由混合分配

## 8. 游戏模式扩展

### 8.1 现有模式保留

| 模式 | 状态 |
|------|------|
| 恶魔塔 | ✅ 保留，层数/房间/怪物不变 |
| 材料秘境 | ✅ 保留，Boss战不变 |
| 训练场 | ✅ 保留，增加DPS统计和技能测试功能 |

### 8.2 新增模式

#### 每日副本（Daily Dungeon）

| 属性 | 说明 |
|------|------|
| 解锁 | 一转后（Lv20+） |
| 次数 | 每天3次 |
| 内容 | 随机3层战斗 → Boss |
| 奖励 | 金币 ×2000 + 技能强化石 ×2 + 洗练石 ×1 |
| 特色 | 每天随机词缀（如"今日怪物+20%HP，掉落+30%"） |

#### Boss Rush 模式

| 属性 | 说明 |
|------|------|
| 解锁 | 二转后（Lv40+） |
| 次数 | 每周3次 |
| 内容 | 连续挑战5个Boss，中途不可回血（每击败一个回复20%HP） |
| 奖励 | 传说装备保底 ×1 + 大量强化石 |
| 排名 | 通关时间排行榜 |

#### 试炼塔（Lv60终局内容）

| 属性 | 说明 |
|------|------|
| 解锁 | Lv60 |
| 内容 | 无限层数，怪物逐渐变强（无上限），每5层一个Boss |
| 奖励 | 神话装备碎片（集齐N个合成神话装备） |
| 重置 | 每周重置一次进度（可购买额外次数） |

### 8.3 模式入口

```

│ 🏰 材料秘境[进入] │

│ ⚡ Boss Rush[3/3] │  ← 新增
│ 🗡️ 训练场  [进入] │
```
