## 一、当前问题

### 1.1 技能列表完全脱离巫师体系

```
巫师技能列表:     ["mage_basic", "elemental_burst", "phase_shift", "resonance_field", "elemental_awakening"]
                    ↑ 相位变换普攻  ↑ 三相位爆发    ↑ 跃迁+位移   ↑ 三元素场     ↑ 三元素觉醒

大魔导师当前列表:  ["mage_basic", "lava_storm",    "aurora_beam",  "blink",        "meteor"]
                    ↑ 旧奥术飞弹  ↑ 纯火球站桩1.5s ↑ 纯冰光束     ↑ 旧闪现       ↑ 纯火陨石
```

**核心矛盾**：巫师建立的三相位轮转、相位脉冲、共鸣/解放/觉醒体系，大魔导师**全部丢弃**，退回到基础法师的单元素静态技能模式。

### 1.2 逐系统诊断

#### 1.2.1 普攻：从三元素弹退化为奥术飞弹

| | 巫师普攻 | 大魔导师"进化"后 |
|------|------|------|
| 形态 | 🔥焚焰弹 / ❄️冰晶镖 / ⚡弧光弹 / 🌈觉醒三合弹 | 旧 `mage_basic` 奥术飞弹 |
| 弹速 | 480/600/700 | 固定 |
| 效果 | 爆炸+灼烧 / 减速+弹跳 / 弹跳+感电 | 无 |
| 3连击终结 | 双倍灼烧 / 双倍减速 / 双弹跳 | 无 |
| 觉醒形态 | 三合弹（灼烧+减速+弹跳） | 无 |

**根本原因**：`wizard-skills-system.js:129-133` 中 `isWizard(player)` 仅返回 `id === 'wizard'`，大魔导师被排除在相位变形系统之外。

```javascript
// wizard-skills-system.js:133
function isWizard(player) {
    const id = classId(player);
    return id === 'wizard';  // ← 仅巫师，大魔导师被排除！
}
```

而 `wizard-element-system.js` 中 `isWizardTree(player)` 却正确包含两者：
```javascript
// wizard-element-system.js:17-19
const WIZARD_TREE = { wizard: true, archmage: true };
function isWizardTree(player) { return WIZARD_TREE[classId(player)]; }
```

这种不一致导致：大魔导师的相位系统在运作（`switchElementPhase` 等函数可用），但技能形态不会随相位变化（普攻、元素爆发等形态覆写被 `isWizard()` 拦截）。

#### 1.2.2 core1：从三相位爆发退化为 1500ms 站桩蓄力

| | 巫师·元素爆发 | 大魔导师·熔岩风暴 |
|------|------|------|
| 前摇 | 🔥300ms 引导 / ❄️瞬发放置 / ⚡瞬发 | **1500ms 站桩蓄力** |

**严格降级**：从 300ms 可移动引导进化到 1500ms 站桩蓄力——这是反向进化。巫师玩家升到 40 级后，操作流畅度大幅下降。

## 二、核心重构理念

**大魔导师不是"更强的巫师"，而是"元素融合者"——在巫师的三相位轮转基础上，掌握相邻相位的双元素熔合。**

### 从三轮转到三桥梁

```
巫师的三相位轮：
     🔥 灼热
    /       \
   /         \
⚡ 过载 ─── ❄️ 极寒
  顺时针自由轮转

大魔导师的三桥梁：
     🔥 灼热
    /   🔥+❄️=熔岩   \
   /                   \
⚡ 过载 ───────────── ❄️ 极寒
   ⚡+🔥=等离子      ❄️+⚡=暴风
   
每次相位切换触发 3 秒"桥接窗口"
桥接窗口内 = 双元素模式
桥接窗口过期 = 回到巫师模式
```

### 核心机制：相位桥梁 (Phase Bridge)

| 属性 | 值 | 说明 |
|------|------|------|
| 触发条件 | 切换相位 | 每次相位切换触发 |
| 持续时间 | **3 秒** | 可被三圣所延长至 5s |
| 内部冷却 | 无 | 每次切换都会重置/刷新桥接窗口 |
| 双元素 | 旧相位 + 新相位 | 窗口内拥有两个相位的能力 |
| 普攻 | 双弹齐发 | 每次发射两发弹（当前 + 前一相位） |
| 核心爆发 | 可长按触发熔合 | 消耗 1 共鸣层数，释放双元素混合法术 |

**设计意图**：
- 巫师 = "数 3 下打出终结弹，存满 4 层共鸣触发解放"（节奏稳定、操作简洁）
- 大魔导师 = "切换后 3 秒内疯狂双弹爆发 + 长按熔合，窗口关闭退守单相位"（节奏跃动、操作分层）

### 数值进化

| 属性 | 巫师 | 大魔导师（新） | 说明 |
|------|:---:|:---:|------|
| 普攻形态 | 3（随相位） | **6**（3单相位 + 3桥接双弹） | 双弹模式=每发0.65×2=1.3x |
| 核心爆发形态 | 3（随相位） | **6**（3轻按巫师模式 + 3长按双元素熔合） | 长按消耗1共鸣层数 |
| 位移 | 跃迁 150px | **维度裂隙 220px + 往返** | 裂隙=空间控制 |
| 团队场 | 3 种（随相位） | **三圣所三合一** | 三元素场叠加 |
| 觉醒 | 6s, +40%/-40% | **8s, +60%/-60%, 击杀延长** | 击杀+1s，最大+4s |
| HP | 0.9 | **0.95** | 移除生存退化 |
| 魔攻 | 1.35 | **1.5** | 强化输出定位 |
| 速度 | 0.95 | **0.95** | 移除速度退化 |
| CD 速度 | 1.0 | **1.05** | 新增小幅度 CD 加速 |

## 三、技能完全重构

### 3.1 slot: basic — 相位双生弹 (Phase Twin Projectile)

从巫师的三相位单弹，进化为**双弹齐发**。

```
属性: homing, 范围 520, CD 500ms, 3 连击链, 连击窗口 2.0s
资源: 元素能量 +8/发 (桥接模式下每对 = 16 能量)

正常相位（无桥接）：
  🔥灼热: 焚焰弹×1 (同巫师，弹速 480)
  ❄️极寒: 冰晶镖×1 (同巫师，弹速 600)
  ⚡过载: 弧光弹×1 (同巫师，弹速 700)

桥接窗口（切换后 3s）：
  双弹齐发 = 当前相位弹 + 上一相位弹
  每发伤害 = 0.65x（总 1.3x），弹速取较快者
  能量: 每对 +16

  切换方向    | 弹A (新相位) | 弹B (旧相位) | 桥接特殊效果
  🔥→❄️ 熔岩  | 冰晶镖      | 焚焰弹       | 双弹命中同一目标 → 触发热冲击(剩余灼烧×2.0结算)
  ❄️→⚡ 暴风  | 弧光弹      | 冰晶镖       | 冰晶弹命中处→弧光弹弹跳以该点为起点
  ⚡→🔥 等离子 | 焚焰弹      | 弧光弹       | 双弹交叉→半径40电弧爆发(0.8x)

3 连击链:
  第1段: 双弹 [0.65×2=1.3x]
  第2段: 双弹 [0.65×2=1.3x]，第二发弹速+15%
  第3段: 双弹 [0.9×2=1.8x]，触发桥接特殊效果（热冲击/弹跳同步/电弧）

觉醒期间:
  三弹齐发 (🔥+❄️+⚡)，每发 0.7x (总 2.1x)
  全部附带灼烧+减速+弹跳
  弹速取三者最快（700）
```

**设计意图**：巫师的操作中普攻是"数 3 下收尾"，大魔导师升级为"切换后 3 秒双弹爆发窗口，在窗口内完成 3 连击链以触发终结效果"。

### 3.2 slot: core1 — 元素熔爆 (Element Fusion Burst)

从巫师"相位决定技能形态"升级为**轻按/长按两种模式**。

```
属性: CD 3.0s, 范围 500+, 消耗 28

轻按 (< 0.4s): 当前相位爆发（同巫师·元素爆发）
  🔥 烈焰风暴: 0.3s蓄力→0.8s可移动引导，6跳×0.5x=3.0x
  ❄️ 极寒新星: 地面放置，0.4s延迟，1.8x+1.2xDoT=3.0x，冻结1.5s
  ⚡ 雷霆之怒: 瞬发，弹跳4次，2.0x→总4.4x

长按 (≥ 0.4s，仅桥接窗口内可用):
  消耗 1 共鸣层数，释放双元素熔合。
  注意：长按要求桥接窗口内且拥有 ≥1 共鸣层数。
  如果桥接窗口外长按 → 退化为轻按模式（无法熔合）。

  🔥→❄️ 熔岩爆发 (Magma Eruption)
    施放: 地面目标点 → 0.6s 后喷发火山柱 + 熔岩扩散
    半径: 130 火山柱 + 熔岩地面持续 4s
    伤害: 直伤 2.5x(火) + 冻结 2s(冰)
          熔岩地面: 每秒 0.6x(火) + 减速 30%
    总伤: 约 4.9x
    消耗: 28 元素能量 + 1 共鸣层数

  ❄️→⚡ 暴风眼 (Tempest Eye)
    施放: 自身中心 → 释放 6 个冰晶向 6 方向扩散 + 连锁闪电
    冰环: 半径 150，1.5x(冰) + 减速 40%
    闪电链: 每个冰晶命中目标 → 触发 1 次闪电链（0.8x，弹跳 2 次）
    最大值: 1.5x + 6×0.8×3×衰减 = 约 6.5x（理论极限）
    实际(4-5怪): 约 4.5x
    消耗: 28 元素能量 + 1 共鸣层数

  ⚡→🔥 等离子波 (Plasma Wave)
    施放: 向前方扇形 120° → 扩散电火冲击波
    范围: 半径 350 穿透波
    伤害: 波浪 2.0x(电) + 灼烧 6s(火)
          波边缘 → 额外 1.0x 电弧爆发
    总伤: 约 4.0x + 灼烧 DOT
    消耗: 28 元素能量 + 1 共鸣层数

觉醒期间（元素化身）:
  长按不消耗共鸣层数
  三元素熔合 → 全屏三色冲击波 4.0x + 全状态(灼烧+冻伤+感电) 6s
  消耗: 仅能量，不消耗共鸣层数
```

**轻按 vs 长按的设计决策树**：
- 轻按 = 巫师模式，无额外消耗，稳定输出 → 新手上手、保守打法
- 长按 = 大魔导师模式，消耗 1 共鸣层数，回报约 +50% 伤害 → 高手爆发、主动决策
- 核心决策：攒共鸣保底触发解放（4层自动消耗）？还是提前消耗 1-2 层换爆发？这形成了"投资 vs 收益"的战略深度

### 3.3 slot: core2 — 维度裂隙 (Dimensional Rift)

从巫师的相位跃迁，升级为**往返裂隙**。

```
属性: CD 6s (比巫师 +1s), 充能 2 层, 消耗 22
传送距离: 220 (巫师 150，+47%)

第一次按下: 裂出
  ├─ 起点留下"裂隙印记"(持续 4s，可见的紫色裂缝特效)
  ├─ 穿越路径伤害: 旧相位 1.0x (巫师是 0.8x)
  ├─ 落点效果: 与巫师相同 (火焰爆发1.2x+灼烧 / 冰盾15%HP 3s / 加速40% 1.5s)
  └─ 同时切换相位 + 触发相位脉冲 + 触发桥接窗口

裂隙持续期间 (4s):
  ├─ 裂隙每 1s 向周围 80 半径脉冲: 0.4x 伤害 + 施加旧相位状态
  ├─ 敌人站在裂隙上 → 持续受 0.2x/s 伤害
  └─ 界面上技能图标变为"返回裂隙"

第二次按下 (裂隙持续期间):
  ├─ 瞬间传回裂隙起点
  ├─ 落点爆炸: 双元素混合 2.0x (旧相位+当前相位)
  ├─ 本段不消耗充能层数（返回是裂隙的"第二段"）
  ├─ 视觉: 裂隙收拢 + 玩家从裂缝中跃出
  └─ 返回后裂隙消失

桥接窗口内跃迁:
  ├─ 路径伤害 = 双元素 1.5x (旧相位 + 新相位)
  └─ 落点效果加倍:
      🔥→❄️: 冰盾吸收 22.5% HP (150%×15%)
      ❄️→⚡: 移速 +60% (150%×40%)
      ⚡→🔥: 火焰爆发半径 105 (150%×70)

消耗规则:
  ├─ 裂出: 消耗 1 层充能
  ├─ 返回裂隙: 不消耗充能（但裂隙在第二次按下后消失）
  └─ 如果裂隙过期(4s): 自动消失，无法返回
```

**战术深度**：
- 巫师跃迁 = 纯功能性（位移 + 切换相位）
- 大魔导师裂隙 = 战术空间控制（📍裂隙起点 + 跃迁输出 + 敌人逼近 → 返回安全区 + 高额爆炸伤害）

**场景应用**：
```
Boss 站位 → 裂隙标记安全角
  → 跃迁到 boss 背后输出（桥接窗口双弹爆发）
  → Boss 转身攻击 → 返回裂隙
  → 安全脱离 + 裂隙爆炸 + 相位切回重新开始循环
```

### 3.4 slot: team — 三圣所 (Tri-Sanctuary)

从巫师的三选一场，进化为**三合一圣所**。

```
属性: CD 14s, 消耗 30, 持续 8s
放置半径: 160 (巫师 120)
放置方式: 鼠标指向位置

效果:
  ├─ 范围内队友同时获得:
  │   ├─ +18% 攻击力       (巫师灼热场: +15%)
  │   ├─ +24% 减伤         (巫师冰封场: +20%)
  │   └─ +30% 攻速/CD速度  (巫师电弧场: +25%)
  │
  ├─ 范围内敌人:
  │   ├─ 每秒施加 1 层灼烧 4s
  │   ├─ 减速 35%
  │   └─ 每 1.2s 自动电击 0.6x 一次
  │
```

### 8.2 wizard-element-system.js — 新增桥梁模式

```javascript
// 新增常量与状态字段
const BRIDGE_WINDOW_MS = 3000;        // 桥接窗口持续时间
const BRIDGE_SANCTUARY_MS = 5000;     // 三圣所内延长至 5s

// 进入桥接窗口
window.enterBridgeWindow = function(player, fromPhase, toPhase) {
    let durationMs = BRIDGE_WINDOW_MS;
    if (player._triSanctuaryActive) {
        durationMs = BRIDGE_SANCTUARY_MS;
    }
    player._bridgeWindowUntil = Date.now() + durationMs;
    player._bridgeFromPhase = fromPhase;
    player._bridgeToPhase = toPhase;
    // 可选：触发 VFX 指示器
    // showBridgeWindowVfx(player, fromPhase, toPhase);
};

// 检查是否在桥接窗口内
window.isInBridgeWindow = function(player) {
    const id = classId(player);
    if (id !== 'archmage') return false;
    return player._bridgeWindowUntil && Date.now() < player._bridgeWindowUntil;
};

// 获取桥接融合类型
// 返回: 'magma' | 'tempest' | 'plasma' | null
window.getBridgeFusionType = function(player) {
    const from = player._bridgeFromPhase;
    const to = player._bridgeToPhase;
    if (from === 'fire' && to === 'frost') return 'magma';
    if (from === 'frost' && to === 'overload') return 'tempest';
    if (from === 'overload' && to === 'fire') return 'plasma';
    return null;
};

// 桥接窗口内获取双相位伤害修正
window.getBridgePhaseModifiers = function(player) {
    if (!isInBridgeWindow(player)) return null;
    return {
        fromPhase: player._bridgeFromPhase,
        toPhase: player._bridgeToPhase,
        fusionType: getBridgeFusionType(player),
        // 双元素伤害各取 60% 修正（而非完整 100%）
        fromDmgMult: 0.6,
        toDmgMult: 0.6
    };
};
```

### 8.5 长按机制 (hold detection)
