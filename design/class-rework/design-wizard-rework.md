# 巫师 (Wizard) 技能完全重构 —— 设计文档

> 版本：v1.0 | 日期：2026-07-11

---

## 一、当前问题

### 1.1 与战士/弓箭手的系统性差距

| 维度 | 战士 (3分支) | 弓箭手 (3分支) | 巫师 (1分支) |
|------|:---:|:---:|:---:|
| T1 技能数 | 9个 (3分支×3) | 9个+ (3分支×3+) | **3个** |
| 专用子系统文件 | 2个 (~14KB) | 6个 (~118KB) | **1个** (~12KB) |
| 位移技能 | 冲锋系列(3个) | 后跳/风步/幻影闪烁 | **0个** |
| 防御机制 | 圣盾/减伤光环 | 闪避/无敌帧 | **仅跑速±** |
| 团队技能 | 战吼/神圣领域 | 标记/致命光环 | **0个** |

### 1.2 逐技能诊断

**烈焰弹 (flame_bolt)：全游戏最长前摇**
- `windupMs: 1200` — 站桩蓄力 1.2 秒，在 ARPG 中完全无法使用
- 弹速仅 380，消耗 28 元素能量，两个火球空蓝
- 无移动取消机制，无蓄力中途打断手段

**闪电链 (chain_lightning)：定位混乱**
- 范围仅 250，法师处于危险的中距离
- 5 连锁衰减 0.15 → 第5目标仅 0.94x，连锁价值极低
- 进化自冰霜新星（冰 → 电突变），玩家丢失冰系技能

**元素协奏 (elemental_symphony)：没有独立身份**
- 1.5x 伤害 + 触发 2 次共鸣 + 进入解放 = 纯"启动器"
- 战士/弓箭手的终极技均有独立视觉和战术身份，巫师没有

### 1.3 系统层面缺陷

| 缺陷 | 详情 |
|------|------|
| **巫师没有冰系技能** | 极寒相位提供 +20% 冰伤 + CD×2，但巫师一个冰技能都没有 → 四相变两相 |
| **共鸣系统 T1 不可用** | thermal_shock 只有大魔导师的 lava_storm 能触发 → 不开终极=共鸣离线 |
| **相位路径锁死** | fire→frost↔fire、overload→fire、arctic→fire，无自主选择权 |
| **零位移/零防御** | 跃迁/闪现去了术士线，巫师没有任何位移或自保能力 |
| **团队槽位空缺** | 基础法师的 blink 进化到术士的召唤骷髅，巫师没有 team 技能 |

## 二、核心重构理念

**巫师不是"放火球的法师进阶版"，而是"元素的舞者"——每一拍切换都在创造新的可能性。**

三大层面重构：
- **相位**：从 4 个维度但只用 2 个 → **三位一体循环轮**
- **技能**：从 3 个独立按钮 → **相位 × 技能 = 变换形态**
- **共鸣**：从终极技专用启动器 → **切换即堆叠的持续运营系统**

## 三、相位系统：三位一体轮

### 旧设计

```
灼热(火)→霜寒(冰)→过载(电)→极寒(冰+)
    ↑____________________________↓ 全锁死路径
```

### 新设计

```
      🔥 灼热 (Fire)
     /              \
    /                \
⚡ 过载 (Lightning)───❄️ 极寒 (Frost)
       顺时针自由轮转
```

| 属性 | 旧 | 新 |
|------|------|------|
| 相位数量 | 4 | **3** |
| 切换路径 | 预设死路径 | **顺时针自由轮转** |
| 切换方式 | 施放对应元素技能 | 施放对应技能 **+ 相位跃迁可主动切** |
| 同相重复 | 不可 | **可滞留**，但无共鸣叠层 |
| 相位效果 | 静态数值变化 | **变换普攻 + 变换技能形态 + 数值变化** |

### 三相位效果对比

| | 🔥 灼热 | ❄️ 极寒 | ⚡ 过载 |
|------|------|------|------|
| **角色** | 毁灭者 | 控场者 | 织法者 |
| **增伤** | +25% 火伤 | +20% 冰伤 | +25% 电伤 |
| **资源** | 消耗×1.5 / 回复×0.5 | 消耗×0.6 / 回复×1.5 | 消耗×1.0 / 回复×1.0 |
| **移速** | -10% | +5% | +15% |
| **特殊** | — | CD速度×1.5 | 普攻弹跳 1 次 |
| **普攻** | 🔥焚焰弹 | ❄️冰晶镖 | ⚡弧光弹 |
| **核心技能** | 烈焰风暴 | 极寒新星 | 雷霆之怒 |
| **实用技能** | 跃迁→灼热 | 跃迁→极寒 | 跃迁→过载 |
| **团队技能** | 烈焰场 | 冰封场 | 电弧场 |

## 四、技能完全重构

### 4.1 slot: basic — 相位变换普攻（替换 mage_basic）

普攻形态随相位实时变换。**共享参数**：追踪弹道(homing)、范围 500、回能 8/发、CD 500ms、3 连击链、连击窗口 1.8s。

| 相位 | 普攻名 | 弹速 | 特殊效果 | 3 连击倍率 | 终结强化 |
|------|------|------|------|------|------|
| 🔥灼热 | **焚焰弹** | 480 | 命中爆炸（半径 30） | [1.0, 1.0, 1.3] | 第 3 发双倍灼烧层数 |
| ❄️极寒 | **冰晶镖** | 600 | 命中减速 25%（1.5s） | [1.0, 1.0, 1.3] | 第 3 发减速翻倍(50%) |
| ⚡过载 | **弧光弹** | 700 | 命中弹跳至半径 100 内最近敌人 | [1.0, 1.0, 1.3] | 第 3 发弹跳 2 次 |

### 4.2 slot: core1 — 元素爆发（替换 flame_bolt）

一个技能，三张面孔。**共享参数**：CD 3.0s、范围 500+。

| | 🔥 烈焰风暴 | ❄️ 极寒新星 | ⚡ 雷霆之怒 |
|------|------|------|------|
| **施放方式** | 0.3s 蓄力 → 0.8s **可移动引导** | 鼠标指定位置 → 0.4s 延迟炸裂 | 瞬发 → 弹跳 4 次 |
| **范围** | 前方锥形 300 × 60° | 放置位置半径 110 + 减速场 3s | 初始 600，链跳 250 |
| **伤害** | 6 跳 × 0.5x = **3.0x** | 1.8x 直伤 + 1.2x DoT = **3.0x** | 首目标 2.0x → 衰减 0.12，总约 **4.4x** |
| **特殊** | 每跳叠 1 层灼烧(4s) | 初始冻结 1.5s | 每弹跳目标 +5 能量 |
| **消耗** | 30 | 18 | 25 |

**设计意图**：
- 烈焰风暴 → 解决 1200ms 站桩问题，0.3s 启动后可移动引导
- 极寒新星 → 地面目标型（非自爆），有延迟但带控场
- 雷霆之怒 → 高速弹跳清杂，给过载相位爆发工具

### 4.3 slot: core2 — 相位跃迁（替换 chain_lightning）

既是位移，又是切换，又是爆发。**共享参数**：CD 5s、充能 2 层、消耗 20、传送距离 150。

| 跃迁至 | 终点效果 | 自身增益 |
|------|------|------|
| **→ 🔥灼热** | 半径 70 火焰爆发 (1.2x) + 灼烧 3s | — |
| **→ ❄️极寒** | 半径 70 冻结 (1.0s) | **冰盾**：吸收 15% 最大生命，持续 3s |
| **→ ⚡过载** | 半径 70 感电 (2s) | 移速 +40%，持续 1.5s |

**穿越伤害**：跃迁穿过敌人时，额外造成 0.8x 路径伤害（使用旧相位元素）。

**战术场景**：
```
灼热输出中 → 敌人跳脸 → 跃迁切极寒
  → 冻结近身怪 + 冰盾保命 + 进入冰冻风筝模式

极寒风筝中 → 跃迁切过载
  → 穿怪群路径伤害 + 加速拉开 + 弧光弹弹跳清场

过载游击后 → 跃迁切灼热
  → 火焰落地爆发 + 进入爆发输出模式 + 烈焰场增伤
```

**一石三鸟**：补位移（之前缺失）、补防御（冰盾）、补相位主动切换（之前没有）。

### 4.4 slot: team — 元素共鸣场（新技能，填补空缺）

**共享参数**：CD 12s、消耗 25、持续 6s、放置半径 120、鼠标指向位置。

| 相位 | 场名 | 队友效果 | 敌人效果 | 视觉 |
|------|------|------|------|------|
| 🔥灼热 | **烈焰场** | +15% 攻击力 | 每秒施加 1 层灼烧(3s) | 火焰升腾粒子柱 |
| ❄️极寒 | **冰封场** | +20% 减伤 | 减速 30% + 每 2s 冻结 0.5s | 冰晶护罩若隐若现 |
| ⚡过载 | **电弧场** | +25% 攻速/CD | 每 1.5s 自动电击一次(0.5x) | 电弧在地面弹跳 |

**战术场景**：Boss 战放灼热场全队增伤 / 高压波次放冰封场群体减速+生存 / 清怪爆发放过载场加速节奏。

### 4.5 slot: survival — 元素觉醒（替换 elemental_symphony）

```
CD: 50s | 消耗: 100 | 持续: 6s

激活瞬间：全屏冲击波 1.5x 伤害 + 刷新全部技能 CD
觉醒效果：
  ├─ 进入"觉醒"相位（不属于三元素，所有相位效果叠加）
  ├─ +40% 全伤, -40% CD
  ├─ 普攻三合一：同时附带灼烧 + 减速 + 弹跳
  ├─ 元素爆发三合一：
  │   └─ 同时释放：前方锥形烈焰喷射 + 目标点冰晶炸裂 + 全屏连锁闪电
  ├─ 相位跃迁三合一：
  │   └─ 终点同时火焰爆发 + 冻结 + 感电（无需选择相位）
  ├─ 元素共鸣场三合一：
  │   └─ 烈火场 + 冰封场 + 电弧场 三色叠加
  └─ 视觉：全屏微震 + 三色光环旋转 + 金色大字"觉醒!"
```

**不再是启动器，而是一键全开的过载模式。** 觉醒期间巫师化身元素化身——所有技能三元素并存。

## 五、共鸣 & 解放系统重构

### 旧设计

```
触发条件：特定技能 → 计数+1 → 满 3 → 解放
问题：T1 巫师没有触发手段 → 系统离线
```

### 新设计：相位轮转即共鸣

```
每次切换相位 → 共鸣层数 +1（最大 4 层）
每次切换 → 自动触发"元素脉冲"：
  ├─ AoE 伤害（半径 80，1.0x 法强）
  ├─ 新相位状态施加（灼烧 / 冻结 / 感电，2s）
  ├─ 浮动文字 + 层数指示器更新
  └─ 能量返还 10

每层共鸣被动：
  +6% 全伤 | +5% 资源回复速度

满 4 层时：
  下次施放元素爆发 → 自动触发"元素解放"（4s）
  解放效果：+40% 伤害 | -40% CD | 消耗共鸣全部层数
  视觉：巨大双色光环爆炸 + 金色大字"解放!"

层数衰减：
  8s 内未切换相位 → 每秒失去 1 层
  （鼓励持续轮转，惩罚龟缩一个相位）
```

### 新旧对比

| | 旧·律动/共鸣/解放 | 新·共鸣轮转 |
|------|------|------|
| 触发条件 | 特定技能 → 计数 | **切换即+1** |
| T1 可用性 | ❌ 需终极技启动 | **✅ 从第一秒开始** |
| 玩家主动性 | 被动等待触发 | **主动轮转堆叠** |
| 解放频率 | 每 50s 一次 | **约 12-15s 一次** |
| 风险/回报 | 无风险 | **层数衰减 = 有压力** |

## 六、完整操作循环

### 标准战斗（小怪）

```
                         共鸣层数

灼热开局 → 焚焰弹 x2 + 烈焰风暴           0
跃迁切 ⚡ → [脉冲!] 共鸣 1 层 + 穿越伤害    1
  弧光弹弹跳清杂
跃迁切 ❄️ → [脉冲!] 共鸣 2 层 + 冰盾       2
  冰晶镖减速风筝 + 极寒新星冻怪群
跃迁切 🔥 → [脉冲!] 共鸣 3 层 + 火焰落地    3
  烈焰风暴 → 触发解放! 共鸣 4 层
  → 4s 解放窗口：+40% 伤 / -40% CD

解放结束 → 共鸣归零 → 重新积累
```

**循环周期**：约 12-15 秒一次解放爆发

### Boss 战

```
烈焰场(增伤) → 轮转堆叠共鸣 → 解放爆发窗口
   → 4s 内狂按元素爆发 + 跃迁切相位刷脉冲
   → 觉醒(三合一收尾) → 下一个循环
```

## 八、新旧对比总结

| 维度 | 旧·巫师 | 新·巫师 |
|------|------|------|
| **核心技能数** | 3（烈焰弹 + 闪电链 + 协奏） | **4**（元素爆发×3形态 + 相位跃迁 + 共鸣场 + 觉醒） |
| **实际可用形态** | 3 个独立技能 | **10 种**（3形态×3 + 觉醒形态） |

| **位移能力** | 0 | **相位跃迁（2 层充能，5s CD）** |
| **防御能力** | 仅 ±10% 移速 | **冰盾 + 冻结控制 + 跃迁闪避** |

| **共鸣触发** | 离线（需终极） | **持续在线**（切换即触发） |
| **解放频率** | 约 50s 一次 | **约 12-15s 一次** |
| **烈焰弹前摇** | 1200ms 站桩 | **300ms 可移动引导**（烈焰风暴） |
| **终极技身份** | "启动解放的按钮" | **全元素合一·过载模式** |
| **普攻** | 不变的奥术飞弹 | **相位变换的三元素弹** |
