# 死眼 (Deadeye) 技能重设计 —— 设计文档

> 版本：v3.0 | 日期：2026-07-10 | 前置文档：`design-marksman-rework.md`

---

## 一、设计前提

### 1.1 游戏环境约束

| 前提 | 说明 | 设计影响 |
|------|------|---------|
| 复杂多变的地图 | 恶魔塔地图将从方形空间扩展为复杂地形 | 狙击手可以有位置博弈，不需要"被迫站桩" |
| 完美闪避几乎不存在 | 仅风行者风之步的 150ms 窗口可触发，概率≈0 | 不能将"完美闪避"作为核心机制的触发条件 |
| 多人团本即将接入 | team 槽位技能需要为真实队友设计 | 标记/光环类技能的团队增益需要真实落地 |

### 1.2 核心 fantasy

> **"锁定目标后一击必杀的终结者"**——发现弱点、屏息凝神、架枪狙杀。

Marksman → Deadeye 的成长弧线：
- **Marksman（Lv20）**：学会弹药管理 + 精准层数（消耗型），远程可靠输出
- **Deadeye（Lv40）**：学会**维持高位精准 + 屏息窗口 + 狙击终结**，从"消耗型"转变为"维持型"

## 二、当前问题

### 2.1 Deadeye（Lv40）完全未实现

| 维度 | 当前状态 | 评级 |
|------|---------|------|
| `class-config.json` | ❌ 无 resource、无 statsModifier、无 skillEvolution | 🔴 |
| 技能 `perfect_shot` | ❌ 资源类型错（focus 32）、未接入精准层数 | 🔴 |
| 技能 `death_mark` | ❌ 资源类型错（focus 35）、未接入精准层数 | 🔴 |
| 技能 `executioner_gaze` | ❌ 资源类型错（focus 100）、未接入精准层数 | 🔴 |
| 与 Marksman 的衔接 | ❌ 完全断层，弹药+精准系统在 40 级直接崩了 | 🔴 |

### 2.2 Marksman（Lv20）残留问题

| 问题 | 说明 |
|------|------|
| 后跳生存能力弱 | 100 码距离 + 无无敌帧，作为唯一自保手段过于单薄 |
| 弹药 DPS 断层 | 自动换弹不可控，换弹期间只能普攻，手感差 |
| Break 参与度低 | breakDamageMultiplier 全是 1.0x，但 battleRole 是 striker |
| 技能间协同性弱 | 精准层数只有"攒→花"线性逻辑，缺少反馈循环 |

## 三、修正方案

### 3.1 资源统一

| 层级 | 资源类型 | 上限 | 回复 | 说明 |
|------|---------|------|------|------|
| Marksman（Lv20） | ammo | 20 | 0（自动换弹） | 不变 |
| **Deadeye（Lv40）** | **ammo** | **24** | 0（自动换弹 + 屏息期间暂停） | 上限提升 |

### 3.2 Deadeye statsModifier

```json
"statsModifier": {
  "hp": 0.85,
  "attack": 1.35,
  "defense": 0.8,
  "magicAttack": 0.75,
  "magicDefense": 0.85,
  "speed": 0.85,
  "critical": 1.5,
  "dodge": 0.8
}
```

**设计意图**：极致玻璃大炮。攻击 +35%、暴击 +50%，换取 HP -15%、防御 -20%。生存全靠走位和机制。

### 3.3 技能路径进化

```json
"skillEvolution": {
  "slot": "core2",
  "first": "aimed_shot",
  "second": "weakness_mark_de",
  "teamSlot": "team",
  "teamFirst": "weakness_mark",
  "teamSecond": "breath_hold",
  "survivalSlot": "survival",
  "survivalFirst": "death_gaze",
  "survivalSecond": "deadeye_snipe"
}
```

**注意**：Deadeye 的 core2 不再是瞄准射击的升级版——它是一个全新的技能。Marksman 的 `aimed_shot` 转职后被替换为 `weakness_mark_de`（弱点标记·死眼）。

## 四、精准层数机制：Marksman vs Deadeye 分层设计

### 4.1 两种截然不同的玩法逻辑

| 维度 | Marksman（Lv20） | Deadeye（Lv40） |
|------|-----------------|----------------|
| 设计逻辑 | **消耗型**：攒层→花层→爆发→重新攒 | **维持型**：快速叠满→维持高位→被动常驻 |
| 节奏曲线 | 锯齿形（峰值-低谷交替） | 爬坡形（逐步攀到顶后稳定） |
| 操作表达 | "现在花还是再攒一层？" | "我能在屏息期间叠满 15 层吗？" |
| fantasy | 每一发都是精心计算 | 进入专注状态后箭无虚发 |

### 4.2 数值配置

| 参数 | Marksman | Deadeye |
|------|---------|---------|
| 层数上限 | **5**（不变） | **15** |
| 每层攻击力 | +6% | **+4%** |
| 每层暴击率 | +4% | **+2%** |
| 每层暴击伤害 | — | **+4%** |
| 满层总收益 | +30% 攻 / +20% 暴击率 | **+60% 攻 / +30% 暴击率 / +60% 暴伤** |
| 层数消耗 | ✅（瞄准射击 -1 / 死神之眼 -5） | **❌ 永不消耗，永久常驻** |
| 层数衰减 | 无 | 无 |

**设计意图**：Marksman 需要在"花层爆发"和"攒层保底"之间做出决策；Deadeye 没有这个决策——他的全部操作围绕"如何快速叠满 15 层"。15 层上限配合永不消耗，让 Deadeye 的开局节奏非常不同：前 10-15 秒是爬坡期，叠满后面临的是另一套决策（是否此时开屏息？是否开狙击？）。

### 4.3 Deadeye 精准层获取途径

| 来源 | 层数 | 说明 |
|------|------|------|
| 弱点标记命中 | +2 | 稳定 8s CD 来源 |
| 后跳射击使用 | +1 | 机动换取资源 |
| 每 2 次普攻命中 | +1 | 持续输出稳定补充 |
| **屏息期间：以上所有 ×2** | 翻倍 | 屏息 = 快速爬坡窗口 |
| 队友对标记目标暴击 | +1（每 3s 最多 1 次） | 多人补偿 |
| 击破韧性条 | +2 | 破防奖励 |

### 4.4 收益曲线对比

```
Marksman（消耗型，5层）：
  标记 → 2层(+12%攻) → 瞄准(-1) → 1层(+6%攻) → 普攻(+1) → 2层
  → 标记 → 4层(+24%攻) → 死神之眼(-4) → 0层 → 重新开始
  曲线：╱╲╱╲╱╲ （锯齿——峰值极高但会归零）

Deadeye（维持型，15层）：
  标记(+2) → 后跳(+1) → 普攻×4(+2) → 标记(+2) = 7层
  → 屏息! → 标记(+4) → 后跳(+2) → 普攻×4(+4) = 15层（满！）
  曲线：╱╱╱╱╱━━━━━ （快速爬坡 → 持续高位）
```

## 五、完整技能组

### 5.1 core1：后跳射击（继承 Marksman，强化）

| 参数 | Marksman | Deadeye |
|------|---------|---------|
| 弹药 | 8 | 8 |
| CD | 3s | **2.5s** |
| 位移距离 | 140 码 | 140 码 |
| 位移耗时 | 280ms | 280ms |
| 无敌帧 | 前段 120ms | 前段 120ms |
| 箭矢 | 3 发扇形, 总 1.5x | 3 发扇形, 总 1.5x |
| 精准层 | +1 | +1（屏息翻倍 → +2） |

**描述**：向后跳跃 140 码（280ms），前段 120ms 无敌帧。同时射出 3 发扇形箭矢（总 1.5x 伤害）。使用后获得 1 层精准。

---

### 5.2 core2：弱点标记（Weakness Mark）

> 发射追踪弹命中目标，暴露其弱点，持续 **10s**。

| 参数 | 值 |
|------|-----|
| 弹药 | 6 |
| CD | 8s |
| 弹道 | 追踪 |
| 伤害 | 0.5x |
| 标记效果 | 死眼对该目标攻击伤害 **+30%**（独立乘区） |
| 暴击加成 | 死眼对该目标暴击时，伤害额外 **+15%** |
| 精准层数 | 命中 **+2 层**（屏息翻倍 → +4） |
| 击杀引爆 | 目标死亡：**半径 90 码，1.8x 伤害** + 向周围敌人**传染标记** |
| 空场减 CD | 若场上无被标记敌人，CD **减半**（4s） |
| 视觉效果 | 标记目标头顶出现**红色准星**，所有队友可见 |

**实战意义**：
- 标记是 Deadeye 输出的起点：**独立 30% 增伤乘区**，所有后续攻击都享受
- 击杀传染意味着清怪场景标记会自动扩散
- 空场减 CD 保证开局随时可用

**打桩表现**：覆盖率 10s/8s CD → 接近 100%，打桩 DPS 稳定。

---

### 5.3 team：屏息（Breath Hold）

> 死眼进入屏息凝神状态，持续 **8s**。

| 参数 | 值 |
|------|-----|
| 弹药消耗 | **0** |
| CD | 20s（从屏息结束开始计算） |
| 自身暴击率 | **+40%** |
| 自身暴击伤害 | **+25%** |
| 精准层获取 | **×2**（标记 +4、后跳 +2、普攻每 1 次 +1） |
| 自动换弹 | **暂停**（弹药用尽即止，屏息结束后自动装填） |
| 团队光环 | 自身周围 **300 码** 所有队友 **+15% 暴击伤害** |
| 击杀刷新 | 屏息期间击杀标记目标 → 屏息时间**刷新**（最多刷新 1 次） |
| 视觉效果 | 屏幕边缘暗角 + 呼吸音效 + 队友身上金色微光 |

**实战意义**：
- 屏息 = 输出窗口信号。队友看到死眼开屏息应该配合集火。
- 暂停换弹的双向设计：逼你屏息前攒满弹药，制造"弹药打空屏息提前结束"的紧张感。
- 击杀刷新让清怪时屏息持续更久；Boss 战则是固定 8s 窗口。
- 屏息前 7-8 层 → 屏息中 4s 冲到 15 层 → 后 4s 享受满层被动

**打桩表现**：8s 屏息 + 满弹药(24) → 倾泻 24 发普攻 + 2 次标记 + 2 次后跳 → 屏息结束装弹。覆盖率 8/28 ≈ 28.6%。

---

### 5.4 survival：死神狙击（Deadeye's Snipe）

> 死眼**封印所有技能**，架起专属狙击枪，持续 **8s**（无法中途取消）。

| 参数 | 值 |
|------|-----|
| 弹药消耗 | **0**（每发射击消耗算在射击中） |
| CD | 50s |
| 持续 | 8s（5 发自动射击） |
| 技能封印 | core1 / core2 / team **完全禁用** |
| 移动 | **不可移动**（定身），但获得 **30% 减伤** |
| 普攻替换 | 替换为 **「死神之弹」**（自动循环） |

**死神之弹（5 发）**：

| 属性 | 值 |
|------|-----|
| 射击间隔 | 每 **1.6s** 自动发射 1 发（共 5 发） |
| 伤害 | **8.0x / 发** |
| 暴击 | **必暴击** |
| 命中 | **必中** |
| 破甲 | **无视 40% 防御** |
| 弹药消耗 | 每发 **4 弹药**（5 发共 20，需要提前攒） |
| 精准层数 | **不消耗**，每层提供本技能额外 **+15% 暴击伤害** |
| 命中标记目标 | 伤害 **×1.8**（即 14.4x） |
| Boss 击杀 | 重置**全队所有技能冷却** |
| 视觉效果 | 画面变为黑白（除标记目标红色高亮）+ 超大狙击准星 |

**伤害公式（单发，满层 15，命中标记）**：

```
基础 8.0x × 标记 1.8x × 精准层 15×+15%=+225% 爆伤
= 8.0 × 1.8 × (150% + 60% + 25% + 225%)/150%
≈ 8.0 × 1.8 × 3.07
≈ 44.2x / 发
× 5 发 = 221x 总伤害量
```

**实战意义**：
- "找个安全角落 → 标记弱点 → 开屏息叠满 15 层 → 架枪 → 5 发"
- 8s 完全不能动是极大的风险，30% 减伤让团本中可以被保护
- 小怪战浪费大招（没办法，这就是 Boss 特化的代价）

**打桩表现**：5 发 × 1.6s = 8s 循环。消耗 20 弹药需要提前攒满。50s CD。

## 八、弹药物流周期

### Deadeye 标准 Boss 战

```
【0-2s】  弱点标记(6弹) → 叠到2层 → 剩18弹
【2-5s】  普攻穿插 → 叠层
【5-6s】  后跳射击(8弹) → +1层 → 剩10弹 → 触发换弹
【6-8s】  换弹中（约1.2s自动装填）
【8-9s】  弱点标记(6弹) → +2层 → 约7层
【9-12s】 普攻穿插
【12s】   开屏息（弹药约20发）
【12-20s】屏息 8s：
          - 标记(+4层) → 瞬间 11层
          - 普攻×4(+4层) → 15层满！
          - 后跳(+2层，但已满) → 15层
          - 死神狙击（消耗 20弹）→ 5发毁灭打击
【20s】   屏息结束 → 自动装弹 → 下一轮标记循环
→ 50s 大招好 → 再来
```

### 弹药约束一览

| 组合 | 弹药 | 可行？ |
|------|------|--------|
| 标记(6) + 后跳(8) | 14/24 | ✅ |
| 标记(6) + 标记(6) + 后跳(8) | 20/24 | ✅ |
| 屏息全程（标记+后跳+普攻消耗） | 20-24 | ✅ |
| 死神狙击 5 发 | 5×4=**20** | ✅（需提前攒满） |

**设计意图**：死神狙击需要屏息前满弹药，这意味着玩家必须规划——不能在屏息中乱用技能把弹药打光。

## 九、精准层数满层收益对照

| 层数区间 | 攻击力 | 暴击率 | 暴击伤害 | 相当于 |
|---------|--------|--------|---------|--------|
| 0层（开局） | +0% | +0% | +0% | 白板 |
| 5层 | +20% | +10% | +20% | Marksman 满层水平 |
| 10层 | +40% | +20% | +40% | 强力 |
| **15层（满）** | **+60%** | **+30%** | **+60%** | 终极状态 |

### 死神狙击爆伤加成 × 精准层数

| 精准层数 | 每发爆伤加成 | 5发总爆伤加成 |
|---------|------------|-------------|
| 0层（不开狙击） | +0% | +0% —— 别这么干 |
| 8层（屏息爬坡中） | +120% | +120%×5 |
| **15层（满层）** | **+225%** | **+225%×5** |

**差距**：15 层狙击 vs 8 层狙击 ≈ 伤害翻倍。这就是"必须叠满再开大"的数学依据。

## 十、与其他弓箭手分支对比

| 维度 | 游侠→兽王 | 神射手→死眼 | 风行者→幻影 |
|------|----------|------------|------------|
| 核心机制 | 双指令轮换+猎物标记 | **弹药+精准维持+屏息窗口** | 风之印记+分身 |
| 团队价值 | 宠物副坦+猎网定身 | **全队暴伤光环+弱点标记优先目标** | 高闪避生存+风刃AOE |
| Boss 战 | B | **S**（单体爆发+标记增伤+狙击终结） | A |
| 清怪 | A | C（故意为之，特化代价） | A |
| 操作核心 | 叠标记→引爆轮换 | **叠15层→屏息→狙击** | 走位+分身管理 |

## 十三、设计哲学总结

1. **抛弃"完美闪避"**：游戏中完美闪避几乎不存在，改为**可规划的屏息窗口 + 叠层节奏**，操作者有完全掌控。

2. **发现弱点 → 屏息凝神 → 架枪狙杀**：三个技能形成完整叙事链条。标记是"看见了"，屏息是"准备好了"，狙击是"动手了"。

3. **从"消耗型"到"维持型"的玩法转变**：Marksman 的花层爆发表达的是"在正确的时间用正确的技能"；Deadeye 的叠满 15 层维持表达的是"进入完美专注后天下无敌"——两种 fantasy，两种玩法。

4. **多人设计前置**：屏息团队光环 + 标记优先目标 = Deadeye 是队伍里的"节拍器"，他开屏息全队就知道是输出窗口。

5. **保留弱势**：清怪能力刻意维持 C 级 + 狙击时不可移动，这是 Boss 特化职业必须付出的代价。
