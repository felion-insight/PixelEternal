# 兽王 (Beastmaster) 技能重设计 —— 设计文档

> 版本：v1.0 | 日期：2026-07-09

---

## 一、当前问题

### 转职前后操作无变化

| 槽位 | 游侠 (Lv20) | 兽王 (Lv40) | 差异 |
|------|------------|------------|------|
| core1 | 召唤战狼 ×2 | 召唤兽群（狼×2 + 熊） | 多一只熊 |
| core2 | 投掷冰阱 | 剧毒陷阱 | 换一种陷阱 |
| team | 狩猎指令 | 狩猎指令 | 完全相同 |
| survival | 群狼出击 | 兽群狂暴 | 多两只幽灵狼 |

**玩家转了兽王后操作完全一样**：召宠 → 放陷阱 → 标记 → 开大 → 循环。宠物是自动攻击，陷阱是换皮，缺少一个全新的操作维度。

---

## 二、解决方案：兽群协同

### 核心机制：core1 双指令轮换

core1 不再是单纯的召唤键，而是**召唤 + 轮换制双指令**：

| 按下次数 | 行为 | 效果 |
|---------|------|------|
| 第 1 次 | 召唤兽群 | 2 只战狼 + 1 只战熊（永久，直到阵亡） |
| 之后（轮换 ⬆） | **狼群围攻** | 战狼扑咬目标，各 1.8x，施加 2 层「猎物标记」 |
| 之后（轮换 ⬇） | **巨熊粉碎** | 战熊重击 3.0x，消耗所有猎物标记，每层额外 +0.5x + AOE 溅射（半径 80） |

### 猎物标记机制

- 上限 3 层，每层使宠物对该目标伤害 **+8%**
- 来源：狼群围攻 +2 层
- 消耗：巨熊粉碎一次性引爆，标记清零

### 伤害曲线

```
狼群围攻 → 狼群围攻 → 巨熊粉碎（3层引爆）
  2层        3层        3.0x + 3×0.5x = 4.5x AOE
                         溅射半径 80
```

**游侠**：指令 CD 好了就按，无脑循环
**兽王**：必须数标记 → 判断此时引爆还是再叠一层 → 打错轮换伤害减半

## 三、完整技能组

### core1：兽群统领

| 属性 | 值 |
|------|-----|
| 召唤 | 2 战狼 + 1 战熊（永久） |
| 狼自动攻击 | 继承 45% 攻击，撕裂 3s |
| 熊自动攻击 | 继承 30% 攻击，带嘲讽 |
| 熊存活被动 | 主人减伤 15% |
| 狼群围攻 CD | 3s |
| 巨熊粉碎 CD | 共用同一 CD，3s |
| cost | 25 pet_energy |

### core2：猎网投掷（替代剧毒陷阱）

| 属性 | 值 |
|------|-----|
| 形态 | 投掷猎网到目标位置（范围 300） |
| 效果 | 范围 60，敌人定身 2s |
| 宠物联动 | 猎网内敌人受到宠物伤害 **+30%** |
| 最多同时 | 2 张网 |
| CD | 6s，cost 20 |

### team：兽群咆哮

| 属性 | 值 |
|------|-----|
| 标记 | 目标 8s，宠物对标记目标 **+40% 伤害** |
| 指令加速 | 标记期间 core1 指令 CD 减半（3s → 1.5s） |
| 击杀奖励 | 标记目标死亡 → 下一个标记 **0 消耗** |
| CD | 10s，cost 20 |

### survival：万兽奔腾

| 属性 | 值 |
|------|-----|
| 开场 | 半径 160 冲击波 4.0x |
| 召唤 | 4 只幽灵狼，持续 10s |
| 全宠狂暴 | +100% 伤害 + 免疫控制 |
| 指令强化 | 大招期间 core1 指令 **无 CD + 零消耗** |

## 四、操作对比

### 游侠（之前/之后）
```
召狼 → 指令扑咬 → 等 CD → 陷阱 → 标记 → 指令扑咬 → 大招
```

### 兽王（之后）
```
召兽群 → 狼群围攻(叠2层) → 猎网(定身+增伤) → 咆哮(标记+加速)
    ↓ CD 1.5s
狼群围攻(叠到3层) → 巨熊粉碎(4.5x AOE引爆) → 狼群围攻(重新叠)
    ↓
万兽奔腾 → 狼群围攻→巨熊粉碎→狼群围攻→巨熊粉碎...(0CD连打)
```
