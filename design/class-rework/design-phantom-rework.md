# 幻影 (Phantom) 职业重设计 —— 设计文档

> 版本：v2.0 | 日期：2026-07-10 | 前置文档：`../class-system-design.md`、`design-deadeye-rework.md`

---

## 一、设计前提

### 1.1 核心 fantasy

> **"残影共鸣的幻术大师"**——在战场留下延时残影，每一个残影都是过去自己的回响。敌人看到的不是一个人，而是一支**时间错位的幻影军团**。

Windrunner → Phantom 的成长弧线：

- **Windrunner（Lv20）**：速度快、闪避高，风之印记标记敌人，风暴之眼 AOE 收割。操作核心是「跑位 + 风刃」。
- **Phantom（Lv40）**：残影留下延时复制体，**每个残影会复读玩家释放的技能**。操作核心从「自己打」变为「**本体打 + 残影跟打**」，伤害通过残影数量实现乘法。

### 1.2 设计约束

| 前提 | 说明 | 设计影响 |
|------|------|---------|
| 复杂多变的地图 | 恶魔塔地图将从方形空间扩展为复杂地形 | 残影留在关键位置形成"交叉火力"是玩法核心 |
| 完美闪避几乎不存在 | 仅风之行者的 150ms 窗口可触发，不依赖 | 残影生成不依赖完美闪避，由主动技能触发 |
| 多人团本即将接入 | team 槽位技能需要为真实队友设计 | 虚影印的团队增益 + 残影迷惑机制有团本价值 |
| 分身应是"镜像"而非"召唤物" | 残影不应自动攻击，应复读玩家操作 | 残影的所有伤害输出 = 玩家操作的延时复读 |

## 二、当前问题诊断

### 2.1 致命伤：伤害过低

| 维度 | Windrunner（Lv20） | Phantom（Lv40） | 对比 Marksman→Deadeye |
|------|-------------------|----------------|----------------------|
| attack statsModifier | 1.05x | 1.10x | Marksman 1.50x → Deadeye 1.35x |
| 核心技能基础倍率 | wind_blade 1.0x / 4s CD | phantom_echo_blade 1.0x / 4s CD | aimed_shot 2.5x / 4s CD |
| 属性预算分配 | speed 1.25x / dodge 1.25x | speed 1.35x / dodge 1.35x | Marksman attack 1.50x |

**根因分析**：

1. **攻击系数垫底**：1.10x attack 是所有 DPS 职业中最低的（Marksman 1.50x、Berserker 1.25x、Nightblade 1.30x）
2. **技能倍率严重不足**：风刃 1.0x 基础 vs 瞄准射击 2.5x 基础，差距 2.5 倍
3. **属性预算错配**：大量预算花在 speed（1.35x）和 dodge（1.35x），这两个属性不直接提升伤害
4. **相对 DPS 估算**：Windrunner/Phantom ≈ Marksman 的 **45%**

**修复方向**：不靠堆攻击系数（那会失去高机动特色），而是通过**残影复读实现伤害乘法**。

### 2.2 残影"像召唤物"问题

| 问题 | 现状 | 根本原因 |
|------|------|---------|
| 残影行为 | 留在原地，最多复读 2 次 | 复读次数太少、延迟太久（450ms），体感上像"放了就忘" |
| 视觉表现 | 半透明 + 轻微浮动 | 与玩家角色绑定不够紧密，看不出"正在复制我的攻击" |
| 伤害占比 | 残影复读占比 ~15% | 如果残影贡献不到 30%，玩家感觉不到残影的存在价值 |
| 主动性 | 被动等待复读 | 缺少"我打残影跟打"的实时反馈感 |

## 三、核心机制：残影共鸣系统

### 3.1 设计目标

残影不是召唤物，它是**你在时间轴上的投影**。你打什么，残影就打什么。

```
               时间轴 →
玩家:    [风刃发射]──────[移动]──────[普攻]──────[标记]
残影1:        └─450ms→ [风刃复读]
残影2:                         └─450ms→ [普攻复读]
残影3:                                     └─450ms→ [标记复读]
```

**核心体验**：玩家释放技能 → 短暂延迟后 → 所有活跃残影依次打出相同技能。场上残影越多，同一技能的输出倍增效果越强。

### 3.2 残影属性

| 参数 | 数值 | 说明 |
|------|------|------|
| 最大残影数 | **4**（从3提升） | 最多同时存在 4 个残影 |
| 残影持续时间 | **8s**（从6s提升） | 足够支撑一轮技能循环 |
| 复读延迟 | **350ms**（从450ms降低） | 更快反馈，体感更"同步" |
| 复读伤害比例 | **70%**（从60%提升） | 每个残影复读 = 70% 本体伤害 |
| 复读次数上限 | **无硬上限**（由持续时间约束） | 只要残影活着就会持续复读 |
| 残影闪避加成 | 每个残影 +8% 闪避（上限 4×8% = 32%） | 满残影时高闪避 |
| 视野范围 | 残影的面向角度跟随本体（实时更新） | 确保复读弹道方向正确 |

### 3.3 复读机制细节

| 可复读的技能 | 不可复读的技能 |
|-------------|---------------|
| 普攻（archer_basic） | 残影步本身（不递归） |
| 残影风刃（phantom_echo_blade） | 位移/闪现类技能 |
| 虚影印（phantom_mark） | 领域类技能（风暴本身不产生复读风暴） |
| 后跳射击（backstep_shot）的箭矢部分 | 纯 Buff 类技能 |

**复读流程**：
1. 玩家释放可复读技能
2. 系统遍历所有活跃残影
3. 每个残影在 350ms 后从自身位置发射相同的弹幕（70% 伤害）
4. 弹幕使用残影当前面向角度（= 本体角度）

### 3.4 残影伤害乘法计算

```
假设场上 4 个残影，玩家释放一次残影风刃（1.0x 基础）：

本体伤害：    1.0x × (1 + 加速40%) = 1.4x
残影1 复读：  1.0x × 1.4 × 70% = 0.98x
残影2 复读：  1.0x × 1.4 × 70% = 0.98x
残影3 复读：  1.0x × 1.4 × 70% = 0.98x
残影4 复读：  1.0x × 1.4 × 70% = 0.98x
──────────────────────────────────
总计：        1.4x + 0.98x × 4 ≈ 5.32x
```

**4 残影时，一次风刃 ≈ 5.32x 伤害**，与神射手 2.5x 瞄准射击持平甚至更高。这就是「不靠堆攻击系数，靠残影乘法」的核心逻辑。

### 3.5 残影满收益对照

| 残影数量 | 复读总倍率（单技能） | 闪避加成 | 伤害级别 |
|---------|-------------------|---------|---------|
| 0 残影 | 1.0x | +0% | 严重警告——你没有残影 |
| 1 残影 | 1.0x + 0.7x = 1.7x | +8% | 尚可 |
| 2 残影 | 1.0x + 1.4x = 2.4x | +16% | 开始有感觉 |
| 3 残影 | 1.0x + 2.1x = 3.1x | +24% | 进入状态 |
| **4 残影（满）** | **1.0x + 2.8x = 3.8x~5.3x** | **+32%** | 完全体 |

> 注：3.8x~5.3x 的范围取决于是否有风痕/虚影印的独立乘区加成。

## 四、完整技能组

### 4.1 技能槽位总览

| 槽位 | Windrunner（Lv20） | Phantom（Lv40） | 核心变化 |
|------|-------------------|----------------|---------|
| core1 | 风之步（wind_step） | **残影步（phantom_clone）** | 闪现 + 在起点留下残影 |
| core2 | 风刃（wind_blade） | **残影风刃（phantom_echo_blade）** | 残影同步复读风刃 |
| team | 风之印记（wind_mark） | **虚影印（phantom_mark）** | 残影攻击标记目标增伤 + 迷惑 |
| survival | 风暴之眼（phantom_storm） | **虚空风暴（void_storm）** | 移动留下残影轨迹，残影复读伤害提升 |

### 4.2 core1：残影步（Phantom Step）

> 向前闪现并在起点留下一个延时残影，残影会在你释放攻击技能时以 70% 伤害复读。

| 参数 | Windrunner 风之步 | Phantom 残影步 |
|------|------------------|---------------|
| 资源 | wind_mark 15 | wind_mark 20 |
| CD | 4s | **3s** |
| 位移 | 120 码向前 | **150 码向前** |
| 无敌帧 | 150ms | **200ms** |
| 充能 | 1 | **2 层**（充能 CD 同技能 CD） |
| 残影 | — | **在起点生成 1 个残影，持续 8s** |
| 移速加成 | +50% 移速 4s | +30% 移速 4s（降低但靠残影补偿） |
| 风刃 | 释放时射出 3 道风刃 | **改为残影复读机制，不再附带风刃** |

**描述**：向前闪现 150 码（200ms 无敌帧），在位移起点留下一个残影。残影持续 8 秒，会以 70% 伤害复读你的后续攻击。最多 2 层充能，场上最多 4 个残影。

**实战意义**：
- 残影步是残影的主要来源。2 层充能 = 快速铺 2 个残影开局。
- 残影留在位移起点 = 敌人追击时会经过残影区域，残影复读形成交叉火力。
- 利用复杂地形：在转角、柱子后留残影，形成多角度火力网。
- 与虚空风暴联动：风暴期间残影步也留下残影（见 4.5）。

**视觉效果**：闪现时留下一道紫色残影轨迹。生成的残影是半透明的紫色轮廓（随玩家 GIF 帧同步动画），悬浮在生成位置微微浮动。

### 4.3 core2：残影风刃（Phantom Echo Blade）

> 发射穿透风刃。所有活跃残影会在 350ms 后从各自位置复读此风刃。

| 参数 | Windrunner 风刃 | Phantom 残影风刃 |
|------|----------------|-----------------|
| 资源 | wind_mark 18 | wind_mark 15 |
| CD | 4s | **3.5s** |
| 伤害 | 1.0x | **1.0x** |
| 穿透 | 全部穿透 | 全部穿透 |
| 宽度 | 60 | 60 |
| 射程 | 400 | 400 |
| 加速加成 | +40% 伤害 / 2x 宽度 | +40% 伤害 / 2x 宽度 |
| 印记加成 | 对风印/虚影印目标 +50% | 对风印/虚影印目标 +50% |
| 残影复读 | — | **所有活跃残影在 350ms 后从各自位置发射风刃（70% 伤害）** |
| 风暴免 CD | 风暴期间免 CD 免消耗 | 虚空风暴期间免 CD 免消耗 |

**描述**：发射弧形风刃穿透所有敌人。所有活跃残影会在 350ms 后从各自位置同步复读此风刃（70% 伤害）。加速状态下 +40% 伤害且宽度翻倍。对风之印记/虚影印目标额外 +50% 伤害。

**伤害公式**（4 残影，加速状态，印记目标）：

```
本体:   1.0x × 1.4(加速) × 1.5(印记) = 2.1x
残影1:  1.0x × 1.4 × 1.5 × 0.7 = 1.47x
残影2:  1.0x × 1.4 × 1.5 × 0.7 = 1.47x
残影3:  1.0x × 1.4 × 1.5 × 0.7 = 1.47x
残影4:  1.0x × 1.4 × 1.5 × 0.7 = 1.47x
────────────────────────────────────────
总计:   2.1x + 1.47x × 4 = 7.98x / 3.5s CD
```

**实战意义**：
- 这是 Phantom 的**核心输出技能**，也是残影乘法的主要载体。
- 残影从不同位置发射风刃 → 多角度覆盖 → 清怪效率高。
- Boss 战需要先铺残影再输出：残影步×2 → 残影风刃 → 残影同步开火。
- CD 3.5s 比风刃 4s 短，配合虚空风暴免 CD 可实现快速倾泻。

**打桩表现**：4 残影满铺，残影风刃 3.5s 循环。DPS ≈ 7.98x / 3.5s ≈ 2.28x/s 本体等效输出（远超 1.0x/4s=0.25x/s 的旧风刃）。

**视觉效果**：本体发射紫色弧形风刃，每个残影在 350ms 后依次闪烁并发射紫色风刃（略微变暗）。多道风刃从不同角度汇聚到敌人身上。

### 4.4 team：虚影印（Phantom Mirage Mark）

> 标记目标敌人，赋予全队增伤 + 残影迷惑 + 残影增伤的三重效果。

| 参数 | Windrunner 风之印记 | Phantom 虚影印 |
|------|--------------------|---------------|
| 资源 | wind_mark 15 | wind_mark 15 |
| CD | 8s | 8s |
| 伤害 | 0.5x | 0.5x |
| 弹道 | 追踪 | 追踪 |
| 标记持续 | 8s | **10s** |
| 团队增伤 | +15% 伤害 +10% 暴击率 | **+18% 伤害 +12% 暴击率** |
| 风刃增伤 | +50% | +50% |
| 残影增伤 | — | **残影对标记目标 +40% 复读伤害** |
| 残影迷惑 | — | **被标记敌人有 25% 概率攻击附近的残影而非本体** |
| 击杀爆发 | 90 码 / 1.8x / 击退 | 90 码 / 1.8x / 击退 |
| 击杀减 CD | 风之步 -2s | **残影步 -2s**（更快补充残影） |

**描述**：发射追踪标记弹命中目标，持续 10s。团队对该目标 +18% 伤害 +12% 暴击率。你的残影对该目标复读伤害额外 +40%。被标记敌人有 25% 概率将残影误认为本体进行攻击。击杀时范围风爆并减少残影步 CD 2s。

**「残影迷惑」机制详解**：

```
被标记的敌人每次尝试攻击 Phantom 本体时：
  25% 概率 → 改为攻击最近的活跃残影
  75% 概率 → 正常攻击本体

残影被攻击时不受伤（残影是无敌的）
→ 敌人攻击残影 = 白打 = 实质上的 25% 闪避
```

**实战意义**：
- 残影增伤 +40% 是独立乘区：残影复读 70% × 1.4 = 98%，接近本体伤害
- 残影迷惑是 Phantom 在团本中的独特价值：被标记的 Boss 有概率攻击残影，相当于给队伍增加容错
- 标记优先上给精英/Boss，然后残影风刃吃双重加成
- 空场减 CD：场上无标记时 CD 减半（4s）

**打桩表现**：覆盖率 10s/8s CD → 100%。残影对标记目标的复读伤害从 70% 提升到 98%。

**视觉效果**：目标头顶出现紫色三角标记 + 残影轮廓。迷惑触发时目标攻击方向短暂偏离到残影位置。

### 4.5 survival：虚空风暴（Void Storm）

> 化为虚空旋风，移动留下残影轨迹，风暴结束后残影继续战斗。

| 参数 | Windrunner 风暴之眼 | Phantom 虚空风暴 |
|------|--------------------|-----------------|
| 资源 | wind_mark 100 | wind_mark 100 |
| CD | 50s | 50s |
| 持续 | 6s | **8s** |
| 状态 | 无敌 + 高速移动 + 吸引敌人 | 无敌 + 高速移动 + 穿越地形 + 吸引敌人 |
| 牵引力度 | 40 | **60** |
| 每秒伤害 | 0.8x | **1.2x**（每 0.3s 跳 0.6x） |
| 自动风刃 | 每 0.4s 1 道 / 0.6x / 追踪印记 | **无**（改为残影机制） |
| 残影生成 | — | **每移动 0.8s / 24 码 → 留下 1 个残影（4s 持续，45% 复读）** |
| 残影复读加成 | — | **风暴期间所有残影复读伤害 +20%（即 70%→90%）** |
| 结束爆炸 | 2.5x | **3.0x** |
| 免 CD 风刃 | 风暴期间手动风刃免 CD 免消耗 | **风暴期间残影风刃免 CD 免消耗** |

**描述**：化为虚空旋风 8s（无敌 + 高速移动 + 穿越地形 + 吸引周围敌人）。移动时每 0.8s 或每移动 24 码在身后留下残影（4s 持续，45% 伤害复读）。风暴期间所有残影复读伤害提升至 90%。风暴期间手动释放残影风刃免 CD 免消耗。结束时虚空爆炸 3.0x。

**虚空风暴 - 残影风刃联动**：
```
虚空风暴期间（8s）：
  残影风刃 CD = 0（免 CD）
  每次残影风刃消耗：15 wind_mark
  残影复读伤害：90%（风暴加成）
  
  如果风暴前有 4 个残影 + 风暴中留下 4 个残影 = 最多 8 残影同时复读
  但场上最多 4 个残影（风暴轨迹残影 4s 短持续，与新残影轮换）
  实际最优：保持 4 残影 → 残影风刃 × 3-4 次 × 残影复读 90%
```

**虚空风暴 - 残影步联动**：
```
虚空风暴中释放残影步：
  → 在风暴移动路径的起点留下一个 8s 残影（70% 复读，风暴加成 → 90%）
  → 风暴轨迹残影（4s）+ 残影步残影（8s）混合填充残影位
```

**实战意义**：
- 虚空风暴 = Phantom 的「全场铺满残影」窗口
- 风暴中高速移动 → 留下残影轨迹 → 停下来放残影风刃 → 残影同步倾泻
- 最理想循环：风暴前先铺 2 个残影 → 开风暴 → 移动铺满 4 残影 → 站定狂按残影风刃 → 4 残影 × 90% 复读
- 风暴结束爆炸收尾，残影继续战斗数秒

**打桩表现**：风暴 8s 内，残影风刃免 CD。以 3.5s 间隔计算 ≈ 2 次完整残影风刃 + 残影复读。风暴结束爆炸 3.0x。残影复读 90% 加成。

**视觉效果**：玩家角色变为紫色旋风（带虚空粒子效果），移动轨迹留下紫色残影轮廓。暴风结束时所有残影同时闪烁一次后爆炸。

## 六、操作循环设计

### 6.1 标准 Boss 战节拍（30s 窗口）

```
【阶段 1：铺残影（0-6s）】
  残影步(1) → 在原地留残影1
  残影步(2) → 在原地留残影2
  虚影印 Boss → 团队增伤 + 残影增伤
  残影风刃 → 本体 + 2残影同步开火
  → 残影步(3) 充能好 → 残影3

【阶段 2：残影输出（6-20s）】
  残影步(4) → 4残影满！
  虚影印持续刷新 → 标记目标
  残影风刃 × 4-5 次 → 每次 4 残影复读
  残影步保持刷新 → 残影到期前补

【阶段 3：虚空风暴爆发（20-28s）】
  确认 4 残影在场 → 开虚空风暴
  高速移动铺残影轨迹
  站定狂按残影风刃（免 CD）
  → 4 残影 × 90% 复读 × 3-4 次风刃
  → 风暴结束爆炸 3.0x

【阶段 4：收尾（28-30s）】
  残影逐渐消失
  虚影印标记刷新
  等待下一轮
```

### 6.2 清怪场景节拍

```
【小怪波次】
  虚影印标记前排怪 → 击杀风爆扩散
  残影风刃扫射 → 残影复读多角度覆盖
  残影步调整位置 → 留下交叉火力点
  
  虚空风暴 → 冲进怪群
  → 移动吸引聚集
  → 留下残影轨迹
  → 残影风刃 × N 次（免CD）→ 全场清屏
```

### 6.3 弹药/资源管理

| 操作 | 消耗 wind_mark | 说明 |
|------|---------------|------|
| 残影步 | 20 × 2 充能 = 40（开局） | 开局铺残影消耗大 |
| 残影风刃 | 15 / 次 | 常规消耗 |
| 虚影印 | 15 / 8s | 稳定支出 |
| 虚空风暴 | 100 | 需要提前攒满 |
| 自然回复 | 10/s | 100 风印 10s 回满 |

**管理策略**：开局铺 2 残影（40 风印），自然回复覆盖后续。虚空风暴前确保风印接近满值以最大化免 CD 风刃次数。

## 十一、设计哲学总结

1. **「镜像而非召唤」是 Phantom 的灵魂**：残影不自动攻击，它永远是你上一秒的影子。你打什么，它就打什么。

2. **「伤害乘法而非数值堆叠」**：Phantom 的攻击系数只需要 1.15x——一个残影复读 70% = +0.7x，4 个残影 = +2.8x。伤害靠的是残影数量，不是面板。

3. **「位置博弈」**：残影留在生成位置不动。这意味着 Phantom 需要思考在哪里留残影——敌人会走位、地图有障碍，残影的位置决定了复读弹幕的覆盖角度。

4. **「高操作门槛高回报」**：维持 4 残影不简单（每个 8s，需要持续刷新），但 4 残影满时 Phantom 是全游戏 AOE 天花板。

5. **「虚空风暴是 showtime」**：8s 无敌 + 免 CD 风刃 + 残影复读 90% = Phantom 的最高爆发窗口，也是全队可以围绕的战术节点。

6. **「多人中不可替代的价值」**：虚影印的残影迷惑 25% 是 Phantom 独有的团队生存机制。全队 +18% 伤害 +12% 暴击率使其成为强力增伤辅助。
