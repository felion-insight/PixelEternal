# 术士 (Warlock) 技能完全重构 —— 设计文档

> 版本：v1.0 | 日期：2026-07-11

---

## 一、当前问题

### 1.1 致命 Bug：选敌系统完全失效

```
技能调用链：skill-system.js → applyClassSkillPrimaryEffect
    ↓ 传递 ctx = { monsters, bondTarget: castOptions.lockTarget }

术士代码：applyAgonyCurseSkill(ctx)
    → let target = ctx && ctx.lockTarget;   ← 期望 lockTarget
    → 收到的是 bondTarget                          ← 字段名不匹配
    → ctx.lockTarget 永远为 undefined
    → 退化为 auto-find（找最近的怪）               ← 不可控选敌
```

**三重问题叠加**：

| 层次 | 问题 | 影响 |
|------|------|------|
| `skill-system.js:1067-1068` | `castOptions.lockTarget` 仅对 `classId === 'archer'` 和 `lockTargetCenter` 技能设置 | 术士技能**永远不会得到 lockTarget** |
| `skill-system.js:1759` | 传递 `{bondTarget}`，非 `{lockTarget}` | 即使有目标，字段名也不匹配 |
| `warlock-soul-system.js:189` | 检查 `ctx.lockTarget` | 永远读到 undefined |

**结果**：痛苦诅咒、传染诅咒、灵魂收割**全部不可选敌**，只能命中最近怪物。

**额外 Bug**：`startLifeDrain` 函数签名不接受 `ctx` 参数（线 202），`applyWarlockSkillPrimary` 也不传递 ctx 给生命汲取（线 371），导致生命汲取永远不可指定目标。

### 1.2 逐技能诊断

**暗影箭 (shadow_arrow)：没有操作深度**
- 500ms CD 普攻，1.0x 伤害，单发射弹
- 命中诅咒叠层 / 命中腐蚀 ×1.3 → 两个条件被动，无主动操作空间
- 没有连击链、没有蓄力、没有弹跳，纯平A

**痛苦诅咒 (agony_curse)：DOT 大师没有引爆技**
- 叠满 5 层后只能干等 DOT 跳完 — 无 active payoff
- 引爆逻辑（soul_harvest）锁在死灵法师才会获得
- 叠满 5 层后额外叠层纯浪费，无溢出机制

**生命汲取 (life_drain)：全游戏最危险的技能**
- 3 秒站桩引导，不可移动，不可取消
- 术士作为 0.95 HP / 0.9 防御的脆皮，站桩 3 秒 = 必死
- 伤害仅 0.3x/s，回复仅 3%/tick，回报极低
- 无法选敌（ctx 未传递），只能吸最近目标

**暗黑丰收 (dark_harvest)：射程与身份矛盾**
- 以自身为中心半径 100 + 诅咒半径 100 → 总覆盖仅 200
- 术士是远程 DOT 职业（影子箭 520 射程），终结技却要贴脸
- 需要冲到怪堆里放 → 与远程定位完全背离

**召唤骷髅战士 (summon_skeleton_warrior)：换皮无深度**
- 召唤一个骷髅 → 自动攻击 → 没了
- 与旧版 `summon_skeleton`（线 2701）几乎重复，只是换了资源类型
- 无宠物指令、无诅咒联动、无阵亡效果（术士阶段）

### 1.3 系统层面缺陷

| 缺陷 | 详情 |
|------|------|
| **无灵魂溢出机制** | 碎片满 8 → 继续获取直接浪费；无激励管理碎片上限 |
| **单一 DOT 维度** | 只有痛苦诅咒一种 DOT，无 DOT 间的交互/协同 |
| **腐蚀状态利用不足** | 仅影子箭 ×1.3，应用面极窄 |
| **零位移能力** | 巫师有相位跃迁，术士完全无位移，纯站桩 |
| **旧僵尸技能残留** | `shadow_bolt`(线2666)、`summon_skeleton`(线2701) 仍存在，使用 mana 而非 soul_shard |
| **T1 无可解锁内容** | 3 个技能都是即时可用，无"先解锁再释放"的成长感 |
| **碎片来源单一** | 仅靠击杀获得，长 Boss 战碎片枯竭 |

## 二、核心重构理念

**术士不是"会 DOT 的法师"，而是"织网者"——编织诅咒之网，然后一把火烧光。**

四个核心动词：
- **织网 (Weave)**：在敌人间铺设诅咒层数和腐蚀
- **引爆 (Detonate)**：一口咬碎诅咒之网，将 DOT 化为爆发
- **燃烧 (Burn)**：灵魂满溢时进入过载，DOT 伤害飙升
- **收割 (Reap)**：从死亡中获取资源，正反馈循环

## 三、灵魂碎片系统重构

### 旧设计

```
碎片上限：8
获取：击杀 +1 / DOT 跳 8-15% 概率 +1
消耗：召唤 2 碎片 / 暗黑丰收 全部
溢出：直接浪费
```

### 新设计：三层机制

```
┌─────────────────────────────────────────┐
│ Layer 1: 碎片获取                         │
│  ├─ 击杀 +1                              │
│  ├─ DOT 跳 8-15% 概率 +1 (保持不变)        │
│  ├─ 腐蚀目标死亡 +1 (新!)                 │
│  ├─ 引爆诅咒时每层 +0.2 碎片 (新!)         │
│  └─ 痛苦箭(combo第3段) +1 (新!)           │
├─────────────────────────────────────────┤
│ Layer 2: 灵魂燃烧 (Soul Burn)             │
│  触发：碎片满 8 → 自动进入灵魂燃烧         │
│  持续：8s (期间碎片消耗无效，锁定 8)        │
│  效果：                                   │
│    ├─ 全 DOT 伤害 ×1.5                    │
│    ├─ DOT 跳频 ×2 (每 500ms 跳)           │
│    └─ 暗影箭弹速×1.3，每发带 1 层诅咒      │
│  结束：8 秒后碎片归 0 (燃尽)               │
├─────────────────────────────────────────┤
│ Layer 3: 诅咒引爆 (Curse Detonation)      │
│  5 层诅咒目标 → 可用引爆取而代之释放        │
│  消耗：1 碎片                             │
│  效果：                                   │
│    ├─ 结算目标剩余全部 DOT 伤害(秒数×跳伤)  │
│    ├─ 半径 80 爆炸 (1.5x 魔攻)             │
│    ├─ 爆炸范围内溅射 2 层诅咒               │
│    └─ 目标若死于引爆 → 返还 2 碎片          │
└─────────────────────────────────────────┘
```

### 新旧对比

| | 旧 | 新 |
|------|------|------|
| 溢出处理 | 浪费 | **灵魂燃烧·全 DOT 增益** |
| 引爆手段 | 无 (T1) | **诅咒引爆 (基础技能)** |
| 碎片来源 | 1 种(击杀) | **5 种** |
| 管理节奏 | 攒→用→攒 | **积满→燃烧→引爆→积攒** |
| 风险/回报 | 无 | **溢出不亏、引爆时机重要** |

## 四、技能完全重构

### 4.1 slot: basic — 暗影连箭（替换 shadow_arrow）

不再是单发平 A，而是**3 连击链**。每段伤害/效果不同，第三段有特殊奖励。

| 属性 | 值 |
|------|-----|
| CD | 500ms |
| 连击窗口 | 2.0s（超过则从第一段重新开始） |
| 弹道 | 追踪(homing)，射程 520 |
| 资源 | 无消耗 |

| 段 | 名称 | 倍率 | 弹速 | 弹体颜色 | 效果 |
|------|------|------|------|------|------|
| 第 1 段 | 暗影箭 | 1.0x | 480 | 紫黑 `#663399` | 基础伤害 |
| 第 2 段 | 暗影箭·蚀 | 1.0x | 500 | 深紫 `#552277` | 命中施加腐蚀 4s |
| 第 3 段 | 痛苦箭 | 1.4x | 550 | 亮紫 `#aa44ff` | **穿透**; 命中诅咒目标 +1 碎片; 命中腐蚀目标 ×1.5 伤害 |

**诅咒/腐蚀互动保留**：
- 命中已有痛苦诅咒目标 → 叠加 1 层（仅第 1、2 段）
- 命中腐蚀目标 → 伤害 ×1.3（仅第 1、2 段；第 3 段已是 ×1.5）

**设计意图**：给平 A 一个节奏感——你需要数到 3 打出痛苦箭，然后在 2 秒内打出下一轮的第 1 段。

### 4.2 slot: core1 — 痛苦诅咒 + 诅咒引爆（二合一技能，替换 agony_curse）

```
┌─ 对未满层目标 ──→ 施加痛苦诅咒 (同现行)
│                   效果不变：max 5 层，0.15x/s DOT，8s 持续
│                   消耗：0 碎片，CD 1.5s
│
└─ 对已满 5 层目标 ──→ 诅咒引爆 (新!)
                       消耗：1 碎片，CD 1.5s (共享 CD)
                       效果：
                         ├─ 结算剩余 DOT：剩余秒数 × 0.15x × 5
                         │   例：剩余 6s → 6 × 0.15x × 5 = 4.5x (一次性爆发!)
                         ├─ 半径 80 爆炸伤害 1.5x 魔攻
                         ├─ 溅射：半径 80 内所有敌人施加 2 层诅咒
                         ├─ 目标被杀死 → 返还 2 碎片
                         └─ 视觉：紫黑色能量从目标体内炸开，碎片粒子飞溅
```

**CD 共享的原因**：同一个键位，同一个 CD。防止叠满后同时引爆 + 立即再叠的滥强循环。

**操作逻辑**：
```
对 Boss：叠→叠→叠→叠→叠(满5)→引爆!→再叠→叠→叠→叠→叠→引爆!
对群怪：叠 A 2 层 → 叠 B 2 层 → 叠 C 1 层 → 引爆 A(溅射B和C满层)→引爆 B→引爆 C
```

### 4.3 slot: core2 — 灵魂链接 (替换 life_drain)

**不再引导！** 改为标记型技能。

```
标记目标 8s, CD 8s, 消耗 0 碎片, 射程 500

标记效果：
  ├─ DOT 吸血：标记目标受到的所有 DOT 伤害，30% 转化为术士治疗
  ├─ 碎片生成：DOT 每跳 100% 获得 1 碎片（不再 8-15% 概率）
  │   （= 30% 全 DOT 吸血 + 稳定碎片产出）
  ├─ 标记期间：暗黑丰收对标记目标 ×1.5 结算倍率
  ├─ 标记目标死亡 → 获得 3 碎片 + 标记转移到最近敌人（剩余时间）
  └─ 视觉：敌人头上悬浮紫色灵魂火焰图标 + 吸血线连接

灵魂燃烧期间：
  ├─ 吸血比例 → 50% (↑20%)
  └─ 碎片获取 → 每跳 +2 (双倍)
```

**为什么替代生命汲取**：

| | 生命汲取(旧) | 灵魂链接(新) |
|------|------|------|
| 施放方式 | **3s 站桩引导** | **瞬发标记** |
| 生存能力 | 站桩 = 吃伤害 | **远程标记 = 安全** |
| 吸血量 | 3%/tick × 3 ticks = 9% | **30% × 全 DOT 伤害（远超 9%）** |
| DOT 联动 | 仅腐蚀 ×2 跳频 | **所有 DOT 吸血 + 稳定碎片** |
| 碎片产出 | 完成引导 +2 | **每 DOT 跳 +1（持续产出）** |
| 可取消 | 不可取消 | **瞬发无惩罚** |

### 4.4 slot: team — 召唤暗影魔 (替换 summon_skeleton_warrior)

不再是永久小兵，改为**短时强力召唤**。

```
消耗 2 碎片, CD 12s, 持续 10s, 最多 2 只同时存在

暗影魔属性：
  ├─ HP: 35% 术士最大生命
  ├─ ATK: 50% 术士魔攻
  ├─ 攻击间隔: 800ms
  ├─ 每次攻击施加 1 层痛苦诅咒 (关键!)
  ├─ 优先攻击已有诅咒的目标
  └─ AI: 近战追逐

暗影魔死亡时 (无论到期/被杀)：
  ├─ 展开半径 60 暗影爆炸 (1.0x 魔攻)
  ├─ 范围内敌人施加 1 层诅咒 + 腐蚀 4s
  └─ 暗影魔死亡使术士获得 1 碎片 (新碎片来源!)

灵魂燃烧期间召唤：
  ├─ 暗影魔攻击 ×1.3 伤害
  └─ 每次攻击施加 2 层诅咒 (双倍叠层!)
```

**为什么替代骷髅战士**：
- 暗影魔有**主动诅咒生成**（自动给怪叠层，为引爆铺路）
- 有时限 = 需要规划释放时机（不是无脑召）
- 死亡有 aoe + 碎片回馈（"燃尽"美学的延续）
- 灵魂燃烧时更强（系统间联动）

### 4.5 slot: survival — 暗黑丰收·指向型 (替换 dark_harvest)

不再是自身中心，改为**鼠标指向位置释放**。

```
消耗：全部碎片，CD 30s

释放方式：
  ├─ 玩家点击 pointing 位置 (射程 400)
  ├─ 放置一个半径 120 的暗黑漩涡 (0.4s 延迟)
  └─ 视觉：地面出现紫色符文阵，旋转收缩

结算阶段（漩涡激活时）：
  ├─ 每片碎片结算区域内敌人 2s DOT 伤害
  │   例：8 碎片 → 结算 16s 等效 DOT
  ├─ 区域内敌人全部施加 1 层诅咒
  ├─ 区域内敌人全部施加腐蚀 8s
  └─ 碎片数 ≥ 5 → 额外触发一次诅咒爆炸 (0.5x 溅射半径)

灵魂链接加成：
  └─ 标记目标在区域内 → DOT 结算倍率 ×1.5

灵魂燃烧期间：
  └─ DOT 结算倍率 ×1.5 (叠加 = ×2.25 对标记目标!)
```

**为什么从自中心改为指向型**：
- 术士是远程：520 射程的影子箭 vs 200 覆盖的终极 → 完全矛盾
- 参考巫师共鸣场·地面放置 — 技术路径一致
- "在怪群里丢漩涡"比"冲到怪群里引爆"更有远程操控感

## 五、完整操作循环

### 标准小怪战斗

```
阶段 1: 织网
  暗影连箭 1→2→3（痛苦箭穿透+加碎片）
  丢暗影魔（自动叠诅咒）
  灵魂链接标记精英怪

阶段 2: 引爆
  诅咒满 5 → 核心 1 引爆!
  → 收碎片 + 爆炸 + 溅射诅咒 + 范围伤害

阶段 3: 燃烧（碎片满时）
  灵魂燃烧触发 → DOT 伤害飙升
  → 暗影魔双倍叠诅咒
  → 疯狂引爆-叠-引爆-叠

阶段 4: 收网
  暗黑丰收丢怪群中心
  → 消耗全部碎片结算 DOT + 上诅咒
  → 再次回到阶段 1
```

### Boss 战循环

```
开战：暗影魔 ×2 → 灵魂链接 Boss
积攒：
  暗影连箭循环 (3连击节奏)
  暗影魔自动叠诅咒
  灵魂链接持续产碎片

爆发窗口 (诅咒满 5 层)：
  引爆! (1 碎片) → 溅射 Boss 2 层 → 继续叠 → 引爆!

燃烧窗口 (8 碎片)：
  灵魂燃烧自动触发 (8s)
  → 暗影魔双倍叠层 → DOT ×1.5 → 快节奏引爆
  → 8s 后碎片归零 → 重新积攒

终场：
  暗黑丰收丢 Boss 脚下 → 8 碎片全结算 → 返回积攒
```

## 六、与死灵法师 (Necromancer) 的进化预览

| 术士技能 | → | 死灵法师技能 | 进化点 |
|------|------|------|------|
| 暗影连箭 | → | **死亡缠绕** | 第 3 段变为双发 (2 枚痛苦箭); 命中 ×2 碎片 |
| 诅咒引爆 | → | **灵魂收割** | 引爆后诅咒不清零(降至 2 层); 额外补充自身 15% HP |
| 灵魂链接 | → | **亡者印记** | 可持续 12s; 标记目标死亡→召唤一个临时骷髅(15s) |
| 暗影魔 | → | **亡灵军团** | 消耗所有碎片, 每 2 碎片召唤 1 只骷髅 (战场可铺 4-7 只) |

### 死灵法师核心演变

```
术士 = 精致操作 (1-2 个暗影魔 + 精确引爆)
     ↓
死灵法师 = 亡灵海 (6-7 只骷髅 + 不停爆炸 + 传染诅咒)
```
