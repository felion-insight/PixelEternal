## 深阶装备（360 件）

### 命名规则

形如 **`{主题}{凡|良|湛|炽|曜}·{后缀}`**。

- **后缀**来自 `config/deep-suffix-table.json`：按 **部位线**（melee / ranged / helmet …）× **主题等级档**（0=渊隙…7=终焉，对应 Lv.25～60）× **品质**（凡→曜 = 索引 0～4）取 **两字名**，同一后缀全局唯一。
- **词条**：由 `js/data-classes.js` 的 `DEEP_TRAIT_FAMILY_BY_LINE` 决定 `void_*` 家族；**主题**决定档位数 `_0`～`_7`；品质字须与后缀表中的品质列一致，否则游戏内无法解析。

| 部位线 | 部位 | 词条家族（省略 void\_ 前缀） |
|--------|------|--------------------------------|
| `melee` | 武器·近战 | sigil, rend, mire, fork, cull |
| `ranged` | 武器·远程 | snipe, ricochet, weaken, volley, star |
| `helmet` | 头盔 | aegis, bastion, pulse, mirror, last |
| `chest` | 胸甲 | spike, dampen, echo, riposte, bulwark |
| `legs` | 腿甲 | surge, grit, ram, strike, overdrive |
| `boots` | 足具 | chase, rush, trace, flash, surge |
| `necklace` | 项链 | mend, skill, snare, arc, well |
| `ring` | 指环 | twin, sever, fervor, tempo, greed |
| `belt` | 腰带 | tithe, hoard, covet, elite, fortune |

### 深阶套装要点

- 散件归属由 `config/set-deep-config.json` 中的 `pieces` 列表判定；**2 / 4 / 6 / 8 件**按档位叠加属性与机制（与常规套装同一套结算）。
- **同一等级主题下，凡品至传说五档的 8 件套机制互不相同**（8 主题 × 5 品质 = 40 种），并非仅数值缩放；完整条目见文末 **「套装效果（深阶）」**。
